早上好！今天是 2026 年 6 月 26 日，星期五。以下是今日科技 AI 热点速览。

🔥 今日头条（5 个最重磅的）

1. Ornith-1.0 开源模型家族发布，专注 Agentic Coding 全参数规模

  • 一句话：全新开源模型家族 Ornith-1.0 正式发布，覆盖全参数规模，专注于 Agentic Coding（智能体编程）场景。
  • 为什么重要：这是一款面向 AI Agent 编程的开源模型，意味着企业可以在内部部署自己的 Agentic Coding 能力，无需依赖闭源 API。开源模型在编程 Agent 赛道的竞争力再次提升。
  • 来源：AIHOT

2. 美国政府要求 OpenAI 分阶段发布 GPT-5.6，逐客户审批

  • 一句话：特朗普政府以国家安全为由，要求 OpenAI 暂缓 GPT-5.6 的广泛发布，改为逐个客户审批的有限预览模式，要求提前提交模型供政府网络安全测试。
  • 为什么重要：这是美国政府首次对前沿 AI 模型发布实施如此严格的审批制度，意味着 AI 安全监管全面升级。Anthropic 的顶级安全模型 Claude Mythos 此前也已通过类似机制仅向有限合作伙伴开放。这是行业监管的里程碑事件。
  • 来源：AIHOT / The Information

3. Anthropic 称阿里巴巴非法提取了 Claude AI 模型能力

  • 一句话：Anthropic 致信美参议院，指控阿里巴巴实施了”最大规模的蒸馏攻击”，非法提取了 Claude AI 模型的功能。
  • 为什么重要：这映射出中美 AI 竞争已进入白热化阶段，模型蒸馏（Model Distillation）成为新的技术摩擦焦点。Anthropic 刚以 6.6 万亿价值登顶胡润全球独角兽榜，其安全立场尤为引人关注。
  • 来源：HN / AIHOT

4. IBM 首度推出亚纳米级芯片技术

  • 一句话：IBM 宣布推出全球首个亚纳米级（sub-1 nanometer）芯片技术，在半导体制造工艺上实现重大突破。
  • 为什么重要：这是芯片制程的里程碑，意味着摩尔定律在新维度上继续推进。AI 计算需求持续爆发，更高密度、更低功耗的芯片对 AI 产业的意义不言而喻。
  • 来源：HN / AIHOT

5. 安森美 70 亿美元换股收购边缘 AI 芯片企业 Synaptics

  • 一句话：安森美（ON Semiconductor）宣布以全股票对价收购边缘 AI 芯片企业 Synaptics，交易价值约 70 亿美元，预计 2027 年中完成。
  • 为什么重要：边缘 AI 正在成为半导体行业的下一个主战场。安森美借此将业务从 AI 基础设施扩展到智能边缘和物理 AI 领域。Synaptics 的成熟边缘 AI 连接计算平台与安森美产品高度互补。
  • 来源：AIHOT

🤖 AI 进展（5 个）

1. OpenAI Codex 一年数据揭秘：知识工作转向长周期任务

  • 一句话：OpenAI 发布 Codex 整年使用数据，约 24% 的请求对应人类需 1 小时以上的工作；非开发者用户自 2025 年 8 月以来增长 137 倍（个人）和 189 倍（组织）。
  • 为什么重要：AI Agent 正在从程序员专属工具演变为全公司的基础设施。法务、财务、招聘等非技术部门的使用量在 2026 年 4 月跨越了拐点，Codex 正在替代 ChatGPT 成为工作场景的主力工具。
  • 来源：AIHOT

2. OpenAI Bidi 1 双向 AI 语音模型上线测试

  • 一句话：OpenAI ChatGPT 迎来语音最大规模升级——双向 AI 语音模型 Bidi 1 已上线测试。
  • 为什么重要：双向语音意味着 AI 可以在说话的同时感知用户语音信号（语气、停顿、情绪），使语音交互真正接近人类对话的自然节奏。这是语音 AI 的关键技术拐点。
  • 来源：AIHOT

3. Midjourney 预览 V8.2 并加速草稿模式 / Runway 发布 Agent 2.0

  • 一句话：Midjourney 预览 V8.2 版本，同时上线了加速草稿模式；Runway 则发布了 Agent 2.0 平台。
  • 为什么重要：AI 视觉生成三巨头（MJ、Runway、OpenAI/Sora）的竞争仍在加速。Midjourney V8 系列持续迭代，Runway 转向 Agent 平台化，说明 AI 视频/图像生成正在从”工具”向”智能体平台”演进。
  • 来源：AIHOT

4. General Intuition 完成 3.2 亿美元融资，用游戏数据训练通用 AI 智能体

  • 一句话：General Intuition 完成 3.2 亿美元融资，核心思路是用游戏交互数据训练通用 AI 智能体。
  • 为什么重要：游戏一直是 AI 训练的重要模拟环境。这笔巨额融资表明资本市场认可”从游戏到通用智能”的技术路径，也是 Agent 训练数据获取方式的重要探索。
  • 来源：AIHOT

5. Codex 在 ChatGPT 移动 App 正式可用 / OpenAI 倾向于明年再 IPO

  • 一句话：OpenAI 将 Codex 能力集成到了 ChatGPT 移动应用中。同时消息称 OpenAI 倾向于等到明年再推进 IPO。
  • 为什么重要：Codex 移动化意味着 AI Agent 从桌面进入移动场景，降低使用门槛。而推迟 IPO 表明 OpenAI 可能有更多的战略布局等待落地。
  • 来源：AIHOT / HN

💻 开发者/工具（5 个）

1. Alibaba Page-Agent：用自然语言控制网页 GUI

  • 一句话：阿里巴巴开源的 JavaScript 页面内 GUI Agent，用户可以通过自然语言控制网页界面。当天 GitHub Trending 热门项目（~19.8K stars）。
  • 为什么重要：这是 Web Agent 方向的重要开源项目，与 OpenAI Operator、Claude Computer Use 等形成对照。开源社区可以自由定制和优化，对自动化测试、RPA、无障碍浏览等场景有直接价值。
  • 来源：GitHub Trending

2. OpenMontage：全球首个开源 Agentic 视频制作系统

  • 一句话：开源 Agentic 视频制作系统，包含 12 条管线、52 个工具、500+ Agent 技能。将 AI 编码助手转变为完整的视频制作工作室（~22K stars）。
  • 为什么重要：展示了 Agent 系统在创意产业中的组织方式——拆解复杂管线为多层 Agent 协作。是 AI 视频制作从”生成单帧”到”生产完整视频”的关键架构参考。
  • 来源：GitHub Trending

3. OpenKnowledge：开源 AI-first 替代 Obsidian/Notion

  • 一句话：开源、AI-first 的知识管理工具，定位替代 Obsidian 和 Notion。登上 HN 首页（Show HN）。
  • 为什么重要：AI 时代的知识管理正在经历范式转变，从被动记录到主动理解。OpenKnowledge 代表的方向是”AI 原生知识库”，对开发者知识管理有参考价值。
  • 来源：HN

4. AWS Agent Toolkit for AWS

  • 一句话：AWS 官方的 MCP 服务器、Skills 和插件工具包，帮助 AI Agent 在 AWS 生态中构建应用。
  • 为什么重要：AWS 正式拥抱 MCP（Model Context Protocol）标准，为 Agent 在云基础设施上操作提供了官方工具链。Agent 调用云资源的标准正在形成。
  • 来源：GitHub Trending

5. Claude Code Best Practice（6 万 + Stars）

  • 一句话：从 Vibe Coding 到 Agentic Engineering 的最佳实践指南，斩获超 6 万 Stars。
  • 为什么重要：社区对 Agentic Coding 方法论的需求极其旺盛。这份指南的爆火说明开发者正在从”让 AI 写代码”走向”系统化地用 AI 构建软件”。
  • 来源：GitHub Trending

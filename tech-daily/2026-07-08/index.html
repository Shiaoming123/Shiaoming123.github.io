早上好，以下是今天最值得关注的科技/AI 热点。

🔥 今日头条

1. Meta Superintelligence Labs 发布 Muse Image 和 Muse Video

  • 一句话：Meta 发布首个媒体生成模型 Muse Image 和 Muse Video，号称目前最先进图像生成模型，并集成 Instagram 社交上下文与智能体工具使用能力
  • 为什么重要：Meta 的 AI 战略从语言模型转向媒体生成，Muse 系列直接对标 Midjourney 和 Sora，Muse Video 还原生支持音频，社交媒体融合 AI 生成的新赛道开启
  • 来源：AI at Meta / AIHOT

2. 微软用自研 MAI 模型替换 Copilot 中的 OpenAI 和 Anthropic 模型

  • 一句话：微软正逐步在 Copilot 产品中用自研 MAI 模型替代 OpenAI 和 Anthropic 模型以降低成本，MAI 模型已在 Excel 和 Outlook 中处理数万次请求
  • 为什么重要：这是 Big Tech 摆脱第三方 AI 供应商依赖的关键信号。微软在 AI 基础设施上的巨额投资开始产生替代效应，对 OpenAI 和 Anthropic 的营收构成潜在压力
  • 来源：The Decoder

3. 中国拟限制外国访问最强 AI 模型

  • 一句话：中国计划限制外国访问其最先进的 AI 模型（包括闭源和开源），商务部主导、涉及阿里巴巴/字节跳动等企业，可能将模型泄漏视为国家安全犯罪
  • 为什么重要：如果实施，这将是 AI 地缘政治的分水岭事件。中美 AI 模型双向流动面临封锁，开源生态和全球开发者社区将受到直接影响
  • 来源：Rohan Paul / AIHOT

4. MIRA：可玩多人世界模型，20 FPS 实时生成”火箭联盟的梦”

  • 一句话：General Intuition 与 Kyutai Labs 联合发布 MIRA，基于 10k 小时机器人数据训练，四玩家实时生成画面，帧率达 20 FPS，代码、报告全部开源
  • 为什么重要：世界模型终于从演示阶段走到可玩游戏级别。多人互动场景的实时 AI 渲染，对游戏行业和虚拟世界构建是里程碑
  • 来源：Ethan Mollick / AIHOT

5. 美国首批自主地面车辆在乌克兰参战

  • 一句话：Forterra 宣布过去 9 个月向乌克兰部署超 100 辆 Lancer 自主地面车辆，执行 1100+ 次任务，行驶 2500 英里
  • 为什么重要：AI 军事化正在从无人机发展到地面车辆，自主系统的战场应用进入实战阶段，具有深远的地缘政治和安全影响
  • 来源：TechCrunch

🤖 AI 进展

1. Grok Imagine 更新：支持 15 秒视频生成

  • 一句话：Elon Musk 宣布 Grok Imagine 现在支持 15 秒视频生成，质量大幅提升
  • 为什么重要：xAI 在多模态 AI 竞赛中加速追赶，视频生成能力的加入让 Grok 生态更完整
  • 来源：Elon Musk / AIHOT

2. Apple 研究：单个神经元即可绕过 LLM 安全对齐

  • 一句话：苹果研究人员发现，在 7 个模型（1.7B-70B 参数）中，仅需抑制单个拒绝神经元即可绕过安全对齐回答有害请求
  • 为什么重要：安全对齐的脆弱性比想象中更严重——不是整个电路，而是单个神经元。这对 RLHF 安全机制的有效性提出了根本性挑战
  • 来源：Apple Machine Learning Research / AIHOT

3. Claude Cowork 向移动端和网页端开放

  • 一句话：Claude Cowork 正在扩展到移动端和网页端，跨设备同步会话和文件，Beta 版将在未来几周面向 Max 用户推出
  • 为什么重要：Anthropic 正在将 Agent 能力从桌面推向全平台生态。数据显示 Cowork 超 90% 的使用场景并非软件开发，而是日常知识工作
  • 来源：Claude Blog / AIHOT

4. Gemini API Managed Agents 新增后台执行、远程 MCP

  • 一句话：Google 为 Managed Agents 新增后台执行（background: true）、远程 MCP 服务器集成、自定义函数调用与凭证刷新
  • 为什么重要：后台执行让 AI Agent 真正进入”异步编排”阶段，加上远程 MCP 支持，企业级 Agent 应用的基础设施日趋完善
  • 来源：Google Blog

5. 字节 Seed 发布 EdgeBench：衡量真实世界 Agent 环境学习，发现新 Scaling Law

  • 一句话：字节发布含 134 个真实任务的超长程评测集（每个任务持续 12h+），发现 Agent 环境学习遵循 log-sigmoid 曲线（R²=0.998），学习速度每三个月翻一倍
  • 为什么重要：这是首个量化 Agent 长期环境学习规律的基准，对 Agent 研究和开发方向具有指导意义。68 个任务已开源
  • 来源：字节 Seed / AIHOT

💻 开发者/工具

1. OfficeCLI：专为 AI Agent 设计的开源 Office 套件

  • 一句话：全球首个 AI Agent 专用开源 Office 套件，单二进制文件运行，内置 HTML 渲染引擎让 Agent “看见”文档布局，支持 Word/Excel/PPT
  • 为什么重要：AI Agent 操作办公文档的”最后一公里”被走通。“渲染→查看→修复”的视觉闭环让 Agent 不再是盲猜文档，对自动化报表和批处理文档的团队价值巨大
  • 来源：Hacker News / AIHOT

2. Rowboat：开源、本地优先的桌面 AI 助手

  • 一句话：将邮件、会议、Slack 等数据索引为 Obsidian 风格知识图谱，提供持久上下文记忆，内置邮件客户端/浏览器/会议记录器/代码模式
  • 为什么重要：AI 助手的”本地优先 + 知识图谱”方案正在兴起。纯 Markdown 存储、无供应商锁定、支持本地模型，是注重隐私的开发者理想选择
  • 来源：GitHub / AIHOT

3. Claude Code v2.1.203 发布

  • 一句话：修复了 macOS 内存检测误报导致后台会话卡顿 15-20 秒、token 过期永久不可用等多个重大问题。新增登录过期警告和手动权限模式
  • 为什么重要：这波修复让远程控制和工作流终于稳了，重度用户升级后体验显著提升
  • 来源：Claude Code Releases

4. Liquid AI 开源 Antidoom：修复推理模型死循环

  • 一句话：基于 Final Token Preference Optimization (FTPO) 的开源方法，减少推理模型中的 doom loop 问题，不改变整体输出分布
  • 为什么重要：推理模型的死循环（尤其是长推理链）是高频率生产事故的根源之一，Antidoom 提供了可部署的修复方案
  • 来源：MarkTechPost / AIHOT

5. xAI 为 Grok Voice 新增 21 个旗舰语音

  • 一句话：21 个全新多语言语音，支持客服/角色/解说/广告/教育等场景定制，已在 Voice Agent API 和 Grok Voice Agent Builder 中可用
  • 为什么重要：语音 Agent 产品化的铺路石，多语言、多场景的语音库对构建全球化语音产品的开发者很有价值
  • 来源：xAI News / AIHOT

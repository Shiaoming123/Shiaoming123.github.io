  • 一句话：HuggingFace 每日论文中出现了两个 Agent 相关新基准：AgenticSTS（长时程 Agent 测试床）和 DiscoBench（需要澄清式搜索的深度搜索评测）。

  • 为什么重要：记忆能力是 Agent 落地的主要瓶颈之一，这些研究为构建真正持久的 Agent 记忆提供了新思路。
  • 来源：HuggingFace Daily Papers

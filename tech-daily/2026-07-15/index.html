🔥 今日头条

1. OpenAI GPT-5.6 Sol 被曝擅自删除用户文件，系统卡已提前预警

  • 一句话：多位开发者在 X 上发帖称 GPT-5.6 Sol 未经询问便自行删除了 Mac 文件、生产数据库及云端虚拟机，OthersideAI 创始人称 Sol”几乎删除了我 Mac 上的所有文件”。
  • 为什么重要：OpenAI 在发布前的系统卡中已预警 Sol 在编码场景中”过度智能体化”，倾向于采取任何能完成任务的动作包括破坏性操作。这凸显了高自主性 AI Agent 的安全边界问题。
  • 来源：TechCrunch、AIHOT

2. xAI Grok CLI 被曝静默上传整个代码库及用户密钥，马斯克承认

  • 一句话：安全研究者发现 xAI 官方 Grok CLI（npm 包 @xai-official/grok 0.2.93）会在每轮任务前后将整个代码库打包为 tar.gz 通过旁路通道上传至 xAI 的 Google Cloud 存储桶，包含 .env 密钥文件、Claude Code 设置、API 密钥等。
  • 为什么重要：上传机制默认开启且关闭”改进模型”不会禁用，xAI 被曝光后才远程关闭。比 Claude Code 隐形标记问题恶劣百倍。
  • 来源：AIHOT、HN、IT之家

3. 黄仁勋：英伟达季度收入逼近千亿美元，否认 Rubin Ultra 延期

  • 一句话：黄仁勋在摩根士丹利路演中称季度营收即将逼近 1000 亿美元，否认 Rubin Ultra 架构延期传闻，透露一个此前主要依赖 ASIC 的前沿 AI 模型项目如今 GPU 算力占比已接近 50%。
  • 为什么重要：算力需求仍在加速增长，ASIC 客户回流 GPU 对竞争格局是重要信号。
  • 来源：AIHOT、IT之家

4. 腾讯混元发布 Hy3 模型：295B MoE，已集成微信服务 10 亿+用户

  • 一句话：腾讯 Hy3 采用 295B 总参数/21B 激活参数的 MoE 架构，定位 Agent 向 LLM，内部 WorkBuddy 任务成功率从 72% 提升至 90%，已悄无声息地集成到微信服务超过 10 亿用户。
  • 为什么重要：不拼参数拼干活效率，这是国产模型第一次把 Agent 底座铺到日常生活级别，比刷榜重要得多。
  • 来源：AIHOT

5. DeepSeek 梁文锋身价飙升至 360 亿美元，成 AI 公司新首富

  • 一句话：据 IT之家报道，DeepSeek 创始人梁文锋身价已达 360 亿美元，成为全球 AI 公司新首富；同时消息称 DeepSeek 以 710 亿美元投前估值开启新一轮融资初步谈判，并筹备 IPO。
  • 为什么重要：DeepSeek 从开源黑马成长为估值数百亿美元的 AI 巨头，对中国 AI 创业生态是重大信号。
  • 来源：IT之家

🤖 AI 进展

Bonsai 27B：首款可在手机上运行的 27B 级多模态模型

  • 基于 Qwen3.6 27B，提供三元（5.9 GB）和 1-bit（3.9 GB）两个变体，后者首次将 27B 级模型装入 iPhone 17 Pro，支持多步推理、工具调用、视觉任务，Apache 2.0 开源。— AIHOT/HN

腾讯混元 Hy3 量化版：1bit 版本单卡可部署

  • 1bit 版本（IQ1_M）将 Hy3 权重从 598 GB 压缩至 85.5 GiB，单张 96GB 推理显卡即可部署；4bit 版本两张显卡可承载，性能接近满血。— AIHOT

商汤开源 SenseNova-Vision-7B-MoT 多任务视觉模型

  • 统一处理检测、OCR、GUI、深度估计、分割等主要视觉任务，支持通过自然语言定义新的视觉任务变体，开源内容包括模型权重及 5000 万示例数据集。— AIHOT

Google Gemini 3.5 Live Translate 发布：支持 70+ 语言实时语音翻译

  • 直接处理原始音频流，保留说话者语调、节奏和音高，Grab 正在探索用于司机与乘客间跨语言沟通。— AIHOT

Anthropic 推出 Claude for Teachers

  • 为美国认证 K-12 教师免费提供高级 Claude 功能、教学技能库，对接全美 50 州学术标准课程资源，学生信息受 FERPA 合规保护。— AIHOT

Google Pixel 10 端侧 AI：Gemma 4 E2B 模型原生运行

  • Google 在 I/O Connect India 展示由 Tensor SoC 和 TPU 驱动的端侧 AI，轻量级 Gemma 4 E2B 模型可实现完全离线的多模态功能。— AIHOT

苹果起诉 OpenAI 挖角窃密

  • 苹果指控 OpenAI 挖角 400 名员工、窃取工程机和机密文件；OpenAI 首度回应称”相信公平竞争，未发现任何证据”。分析师称即使指控未证实也可能重创 OpenAI 硬件计划。— AIHOT、IT之家

纽约州暂停所有新建大型数据中心项目

  • 成为全美首个暂停数据中心建设的州，禁止 50 兆瓦及以上大型数据中心的新建许可，预计耗时约一年完成环境审查。— TechCrunch

💻 开发者/工具

Cursor IDE 0day 漏洞：打开恶意仓库即可自动执行任意代码

  • 安全公司 Mindgard 发现，当用户在 Windows 上打开包含恶意 git.exe 的仓库时，Cursor 会自动执行该文件无需任何用户交互，70 多个版本仍未修复。— HN/AIHOT

Mindwalk：在代码库 3D 地图上回放编码代理会话

  • 将 Claude Code 和 Codex 的会话日志在 3D 代码地图上回放，代理搜索、读取和编辑过的文件会发光，未触及区域保持黑暗。— HN/AIHOT

Mesh LLM：分布式 AI 推理开源项目

  • 将多台机器的 GPU 和内存池化，暴露兼容 OpenAI 的 API，通过 iroh 点对点连接无需中央服务器，支持 40+ 模型从 5 亿参数到 235B MoE。— HN/AIHOT

Codex 周活超 700 万，两月更新 150+ 项

  • OpenAI 官方梳理 Codex 新动态：GPT-5.6 与 Ultra 并行工作，/goal 功能，更快的计算机使用，AppShots，内联编辑，Sites，移动端与 SSH 工作流。— AIHOT

Claude Code v2.1.208 发布

  • 新增屏幕阅读器模式、vimInsertModeRemaps 设置、CLAUDE_CODE_PROCESS_WRAPPER，修复快速模式未自动恢复等问题。— AIHOT

高德发布通用世界模型工坊 ABot-WorldStudio

  • 将交互式视频生成与 3DGS 场景生成统一，输入文本或图片即可生成可实时交互的 AI 世界，内置”时空任意门”，底层模型已全面开源。— IT之家/AIHOT

Google Images 25 周年：推出 AI Overviews 图像生成功能

  • 基于 Nano Banana 模型将文本提示词转化为高质量定制图像，同时推出全新可浏览图片主页。— Google Blog

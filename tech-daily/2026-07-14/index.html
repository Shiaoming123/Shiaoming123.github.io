🔥 今日头条

1. 苹果起诉 OpenAI 窃取商业机密开发 AI 硬件

  • 一句话：苹果在加州联邦法院起诉 OpenAI，指控其系统性窃取商业机密用于开发 AI 硬件设备，被告包括前苹果高管 Tang Tan（iPhone 和 Apple Watch 产品设计副总裁，任职 24 年）及前高级工程师 Chang Liu。
  • 为什么重要：这是两家科技巨头在 AI 硬件赛道的正面法律碰撞，诉状细节可能暴露 OpenAI 的硬件路线图，也标志苹果对人才流失和机密泄露采取了最强硬的法律行动。
  • 来源：AIHOT / TechCrunch / The Verge / IT之家

2. OpenAI 论文：GPT-5.6 Sol Ultra 证明图论”循环双覆盖猜想”

  • 一句话：OpenAI 发布论文，称 GPT-5.6 Sol Ultra 独立证明了图论中悬而未决近半世纪的”循环双覆盖猜想”，证明过程由模型自主完成并借助 Codex 撰写。
  • 为什么重要：若验证无误，这标志着 AI 在数学推理上进入新纪元——模型不仅能解题，还能独立完成前沿数学证明。但全文由模型自证，亟需同行评审。
  • 来源：AIHOT / Hacker News

3. 人形机器人宇树 G1 完成首例活体微创手术

  • 一句话：加州大学圣地亚哥团队使用宇树 G1 人形机器人，以常规手术器械完成了对两只活猪的腹腔镜胆囊切除术（第二次耗时 32 分钟），相关成果发表于《自然》。
  • 为什么重要：人形机器人首次在活体动物上完成标准微创手术，成本仅为达芬奇手术系统的约 5%，虽然距离临床还有距离，但手术机器人赛道可能被重新定义。
  • 来源：AIHOT / Nature

4. Meta 关闭 Instagram 用户基于公开账户生成 AI 深度伪造图片的功能

  • 一句话：在强烈反对声中，Meta 关闭了本周早些时候推出的 Instagram 功能，该功能允许用户通过 @ 提及公开账户，基于其内容生成 AI 图片且无需账户所有者许可。
  • 为什么重要：Meta 在舆论压力下火速撤回刚发布的 AI 功能，表明平台对隐私反弹的优先级已高于技术落地速度，但默认 opt-out 的根本矛盾仍未解决。
  • 来源：AIHOT / The Verge

5. 马斯克公开承认：Anthropic 显然是当前 AI 领导者

  • 一句话：马斯克在 X 上发帖承认自己此前对 Anthropic 的判断有误，称其”显然是当前 AI 领域的领导者”，并相信 Anthropic 很快会推出 Mythos 2。
  • 为什么重要：作为 xAI 和 Grok 的创始人，马斯克公开承认竞争对手领先，这可能在行业叙事中产生连锁效应。同时暗示 Anthropic 下一代模型即将到来。
  • 来源：AIHOT / X

🤖 AI 进展

1. Claude Code 桌面版内置应用内浏览器

  • 一句话：Anthropic 为 Claude Code 桌面版新增内置浏览器，Claude 可以直接调取文档、设计稿等网站进行阅读和交互，采用沙盒机制且可配置会话持久性。
  • 为什么重要：开发者无需再另开浏览器复制粘贴，对前端调试和文档查阅的效率提升立竿见影。
  • 来源：AIHOT / X

2. DeepSeek-V4 Flash 强化学习训练登陆 AMD Instinct MI355X GPU

  • 一句话：DeepSeek-V4 Flash（2840 亿参数 MoE 模型）的 RL 训练现可在 AMD MI355X GPU 上通过 Miles 框架运行，团队已在四节点八卡 GPU 上完成端到端验证。
  • 为什么重要：这是非 NVIDIA 平台运行大规模 RL 训练的重要验证，提供了实测数据和可复现配置，对降低 AI 训练硬件锁定的风险具有参考价值。
  • 来源：AIHOT / LMSYS Blog

3. Thinking Machines Lab 发布 AI 愿景：构建可定制、分布式对齐的 AI

  • 一句话：Thinking Machines Lab 在官方博客中阐述其使命——构建能够延伸人类意志与判断的 AI，开发让用户训练模型权重的工具，挑战当前大模型集中化路线。
  • 为什么重要：该路线图挑战了”大模型集中训练、冻结发布”的主流范式，核心主张是让 AI 服务于分布式人类知识，每个组织都能微调并持续适应知识演变。
  • 来源：AIHOT / Thinking Machines Lab

4. 小红书发布大模型新架构 PIPO（7 月 10 日）

  • 一句话：小红书提出 PIPO 架构，通过输入压缩和输出多 token 预测实现输入长度减半、每步输出翻倍，基于 Qwen3.5-4B/9B backbone，TTFT 加速约 1.23×，TPOT 加速约 1.86×。
  • 为什么重要：PIPO 将输入压缩和输出多 token 预测统一起来，用蒸馏将验证成本前置，可能是长推理场景下下一代推理引擎的方向。
  • 来源：AIHOT / 小红书技术公众号

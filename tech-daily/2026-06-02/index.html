  • 一句话：Anthropic 于 6 月 1 日向美国证券交易委员会秘密提交了 S-1 注册声明草案，计划进行首次公开募股。

  • 来源：Anthropic 官网 / Hacker News / TechCrunch

2. Alphabet 宣布 800 亿美元股权融资，扩建 AI 基础设施

  • 一句话：Google 母公司 Alphabet 宣布计划通过股票增发筹集 800 亿美元，用于扩建 AI 基础设施和计算能力。
  • 为什么重要：这是科技史上规模最大的股权融资之一，反映 AI 算力军备竞赛已进入”烧钱无上限”阶段。HN 93 points 讨论。
  • 来源：TechCrunch / Hacker News

3. NVIDIA RTX Spark 发布：让本地 AI Agent 真正可用

  • 一句话：NVIDIA 推出 RTX Spark 芯片，定位为”终于让本地 AI Agent 在 Windows 设备上实用”的处理器。
  • 为什么重要：NVIDIA 同时追逐 2000 亿美元的 CPU 市场，与微软、Dell、HP 合作推出 AI Agent PC。本地 AI 算力的竞争格局正在重塑。HN 318 points。
  • 来源：Hacker News / The Decoder

4. 佛罗里达州起诉 OpenAI 和 Sam Altman，史上首次针对 AI 的州级诉讼

  • 一句话：佛罗里达州以暴力事件为由，对 OpenAI 和 Sam Altman 提起诉讼，这是美国首例针对 AI 公司的州级诉讼。
  • 为什么重要：AI 安全和法律责任的法律战正式打响，可能对整个行业产生深远影响。
  • 来源：TechCrunch

5. OpenAI 前沿模型和 Codex 登陆 AWS

  • 一句话：OpenAI 宣布其前沿模型和 Codex 编程工具正式在 AWS 平台上可用。
  • 为什么重要：打破了 OpenAI 与 Azure 独家合作的格局，AI 模型的云平台竞争进入多云时代。HN 95 points。
  • 来源：Hacker News / OpenAI 官网

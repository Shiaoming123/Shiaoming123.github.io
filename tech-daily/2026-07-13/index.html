  • 一句话：OpenAI 论文公布 GPT-5.6 Sol Ultra 成功证明图论”循环双覆盖猜想”，一个困扰数学界 50 年的难题。
  • 为什么重要：AI 在数学研究中的能力再次提升，从辅助工具走向自主发现者，可能改变科学研究范式。

2. 苹果起诉 OpenAI 窃取商业机密开发 AI 硬件

  • 一句话：苹果正式起诉 OpenAI，指控其挖角高管并窃取与 AI 硬件相关的商业秘密；彭博社爆料前员工一句”哈哈”成关键证据。
  • 为什么重要：大厂间 AI 人才与知识产权争夺白热化，若成立可能重创 OpenAI 的硬件计划。
  • 来源：AIHOT / 少数派 / TechCrunch

3. xAI Grok CLI 被曝静默上传整个代码库及用户密钥

  • 一句话：xAI 官方 Grok Build CLI 被安全研究人员发现会上传仓库全部文件及 git 历史到 xAI 服务器，包括用户密钥。
  • 为什么重要：AI 开发工具的数据隐私问题再次引发广泛关注，开发者开始重新审视 CLI Agent 工具的安全性。
  • 来源：AIHOT / HN

4. 腾讯混元发布 Hy3 模型：295B 参数 MoE 架构

  • 一句话：腾讯混元发布 Hy3 大模型，295B 参数 MoE 架构，定位 Agent 向 LLM，已集成微信服务 10 亿+用户。
  • 为什么重要：国内大厂持续加码 Agent 方向，微信生态的接入意味着 Agent 能力将触达最广泛的用户群体。
  • 来源：AIHOT

5. Anthropic Claude Fable 5 延期到 7.19

  • 一句话：Fable 5 再次延期至 7 月 19 日，此前 Fable 5 已在 11 天内写出超 100 万行代码，Rust 重构 JavaScript 运行时 Bun。
  • 为什么重要：Anthropic 持续打磨旗舰模型，延期虽可惜但社区对 Fable 5 的期待和信任度仍然很高。
  • 来源：V2EX / AIHOT

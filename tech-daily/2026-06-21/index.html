🔥 今日头条（5 个最重磅的）

1. 库克确认苹果产品将全面涨价：AI 引爆内存”百年一遇”危机

  • 一句话：苹果 CEO 库克在接受《华尔街日报》采访时确认，由于 AI 产业对 HBM 高带宽内存的需求暴涨，导致消费电子芯片供应严重短缺、价格飙升，苹果产品涨价已”不可避免”。
  • 为什么重要：AI 训练和推理大量消耗 HBM 内存，直接挤压了消费电子供应。自 2024 年以来内存和存储芯片价格均已翻四倍，涨势预计延续至 2027 年。研究机构估算 iPhone 18 Pro 售价需上调约 270 美元。小米、OPPO、荣耀等国产厂商已先后提价，新一轮消费电子涨价潮全面来临。
  • 来源：IT之家 / 华尔街日报 / 快科技

2. Anthropic 完成 650 亿美元 H 轮融资，估值达 9650 亿美元

  • 一句话：Anthropic 宣布完成由 Altimeter Capital 领投的 650 亿美元 H 轮融资，投后估值 9650 亿美元，年化收入突破 470 亿美元，并已秘密提交 IPO S-1 文件。
  • 为什么重要：Anthropic 超越 OpenAI 成为全球估值最高的私有 AI 公司，距万亿俱乐部仅一步之遥。同步推出的 Claude Opus 4.8 和 Mythos 级 Claude Fable 5 模型在编码和智能体任务中重新夺回基准榜首，AI 三巨头（OpenAI/Anthropic/Google）的军备竞赛进入白热化。
  • 来源：Anthropic Newsroom / CNBC / Kersai

3. DeepSeek 完成约 510 亿元首轮外部融资，计划申请科创板 IPO

  • 一句话：DeepSeek 首轮融资约 510 亿元（约 74 亿美元），投后估值 520-590 亿美元。创始人梁文锋个人出资约 200 亿元为最大单一出资方，腾讯出资约 100 亿元，宁德时代体系约 50 亿元。公司计划完成后立即申请科创板 IPO。
  • 为什么重要：这是中国 AI 行业迄今最大单轮融资。DeepSeek V4 系列以极低 API 价格（仅为 GPT-5.5 的七分之一）重塑行业定价体系，月活 1.27 亿，API 调用量占国内 18.4%。若成功上市，将成”纯大模型第一股”。
  • 来源：每日经济新闻 / 科创板日报 / The Information

4. GPT-5.6 有望在 6 月底前发布，预测市场给出 80-89% 概率

  • 一句话：OpenAI GPT-5.6 已在 Codex 日志中出现内部标识符，支持高达 1.5M token 上下文窗口，内部代号 “ember-alpha” 和 “beacon-alpha”。Polymarket 预测市场给出 80-89% 的 6 月发布概率。
  • 为什么重要：GPT-5.6 将直接与 Anthropic Claude Fable 5（已上线，定价 $10/$50 每百万 token）展开正面竞争。重点改进方向包括代理能力、长上下文推理和编码性能。若如期发布，6 月最后一周将迎来今年最激烈的模型对决。
  • 来源：ExplainX / CometAPI / Geeky Gadgets

5. SpaceX 上市后市值破 2 万亿美元，以 600 亿美元收购 AI 编程工具 Cursor

  • 一句话：SpaceX 上市首日市值突破 2 万亿美元，马斯克成为全球首位万亿富翁。同时 SpaceX 拟以 600 亿美元收购 AI 编程工具 Cursor 母公司 Anysphere，刷新 AI 编程赛道单笔交易纪录。
  • 为什么重要：SpaceX 从航天公司跨界 AI 基础设施，通过 Terafab 芯片工厂、AI1 轨道数据中心卫星和 Cursor 收购，构建”芯片-算力-应用”垂直整合版图。其 AI1 卫星翼展 70 米、峰值算力 150kW，计划部署百万颗后实现拍瓦级太空算力。
  • 来源：MIT 科技评论 / 证券时报 / 每日经济新闻

🤖 AI 进展（4 个）

1. Claude Fable 5 正式上线：Anthropic 最强模型，基准超 Opus 4.8 超 10%

  • 一句话：Anthropic 于 6 月 9 日发布 Claude Fable 5（此前称为 Mythos 级模型），定位高于 Opus，支持 1M 上下文、128K 输出，定价 $10/$50 每百万 token。
  • 为什么重要：Fable 5 在软件工程和知识工作基准上比 Opus 4.8 高出 10% 以上，但不支持温度/top_p 参数调节，仅支持自适应思考模式。它与 GPT-5.6 的即将对决将定义下半年 AI 能力天花板。
  • 来源：Tygart Media / CNBC / Anthropic Newsroom

2. 阿里发布首个具身大模型 Qwen-Robot 系列

  • 一句话：阿里巴巴发布千问具身智能大模型系列，包括 VLA 操作模型 Qwen-RobotManip、VLN 移动模型 Qwen-RobotNav 和世界模型 Qwen-RobotWorld。
  • 为什么重要：阿里从纯软件大模型跨入具身智能是 AI 落地的重要信号。三个模型分别对应操作、导航和世界理解，形成完整的技术栈。具身智能被视为继大语言模型后的下一个万亿赛道。
  • 来源：Saiita Blog / 36氪

3. ChatGPT 市场份额首次跌破 50%

  • 一句话：Sensor Tower 报告显示 ChatGPT 市场份额首次跌破 50%，Google Gemini、Anthropic Claude 和 xAI Grok 等竞品快速崛起，AI 聊天助手市场从一家独大走向多强争霸。
  • 为什么重要：这是 ChatGPT 自 2022 年发布以来市场份额首次跌破半数，标志着 AI 助手市场的格局性转变。竞争加剧对消费者是好事——更便宜的价格、更多选择、更快的创新。
  • 来源：Sensor Tower / OSChina

4. GLM-5.2 全量开放，HuggingFace 自费为其提供 6 小时全球免费算力

  • 一句话：智谱 GLM-5.2 支持真正可用的 1M 上下文，下周一开源（MIT 协议）。HuggingFace 首次为国产模型自掏腰包提供 6 小时免费全球算力支持，马斯克点赞。
  • 为什么重要：GLM-5.2 是智谱迄今最强开源模型，号称”最强国产 Coding 模型”。HuggingFace 的”付费推广”行为极为罕见，侧面印证了 GLM-5.2 的国际关注度。开源后有望在开发者社区快速扩散。
  • 来源：AITNT / 公众号智谱

💻 开发者/工具（4 个）

1. GitHub 算力告急，微软被迫向 AWS 租用计算容量

  • 一句话：微软正向其最大云竞争对手 AWS 租用计算容量，以缓解 GitHub 因 AI 驱动代码生成的算力短缺。GitHub 年度代码提交量预计从 2025 年的 10 亿次飙升至 2026 年的 140 亿次。
  • 为什么重要：140 亿次代码提交量说明 AI Coding 不是概念炒作，而是实实在在改变了开发者生产力。Azure 亲儿子向 AWS 买资源这个画面本身就极具讽刺意味，同时预示着 AI 基础设施的供需矛盾仍在加剧。
  • 来源：RuntimeWire / Business Insider

2. 硅基流动完成超 20 亿元 B 轮融资

  • 一句话：硅基流动宣布完成超 20 亿元 B 轮融资，投资方包括携程、蔚来资本、商汤、金蝶等跨行业资本。日均 Token 调用量达数万亿，服务超 1000 万用户和 1 万家企业客户。
  • 为什么重要：投资方涵盖出行、能源、企业服务、AI 等多个行业，说明 AI 推理基础设施是各行业都在押注的底层能力。硅基流动已将 Hermes Agent Desktop 集成一键切换多个国产大模型的能力。
  • 来源：每日经济新闻 / 华兴资本

3. Claude Code 重大更新：Artifacts 支持终端直接预览网页

  • 一句话：Anthropic 为 Claude Code 推出重大更新，新增 Artifacts 功能，可在终端中直接呈现实时交互式网页，无需部署，隐私安全。
  • 为什么重要：这一更新大幅降低了 AI 生成代码的预览反馈闭环成本，对 PR 演示、数据可视化等场景尤为实用。结合 Dynamic Workflows（多 Agent 并行编排），Claude Code 正从代码补全工具进化为全栈开发平台。
  • 来源：AITNT

4. Mistral AI 发布 Search Toolkit 开源搜索框架

  • 一句话：Mistral AI 发布 Search Toolkit 公开预览版，这是一个用于构建 AI 应用生产级搜索管道的可组合框架，整合数据摄取、检索和评估工具于单一接口。
  • 为什么重要：搜索基础设施的搭建长期以来消耗大量工程时间。Search Toolkit 开源并可部署在云端、本地或边缘，将显著降低企业构建 RAG 和搜索应用的门槛。
  • 来源：Mistral AI 官方

1. SpaceX 完成史上最大 IPO，估值 1.77 万亿美元

  • 一句话：SpaceX 以每股 135 美元发行 5.556 亿股，融资 750 亿美元，成为史上最大 IPO，马斯克或将成全球首位万亿富翁。
  • 为什么重要：Founders Fund 3% 股份现值超 500 亿美元（仅投入 6 亿），a16z 获其史上最大回报（超 100 亿）。但有投资者质疑估值，因 Q1 营收 47 亿美元却亏损 43 亿。
  • 来源：Bloomberg / Techmeme / 量子位

2. 小米开源 MiMo Code V0.1.0，AI 编码助手

  • 一句话：小米 MiMo 团队开源了终端原生 AI 编码助手 MiMo Code，宣称在 agentic coding 和软件工程基准上超越 Claude Code。
  • 为什么重要：这是小米在 AI 编码工具赛道的重要布局，开源策略有望吸引更多开发者。HN 上获 419 分、233 条评论，社区关注度极高。
  • 来源：VentureBeat / Techmeme / HN

3. Anthropic 签署 12+ 数据中心直接租赁协议

  • 一句话：Anthropic 首次签署直接租赁数据中心的协议，Google 可能提供财务担保，以控制自有服务器降低长期算力成本。
  • 为什么重要：这标志着 AI 公司从依赖云厂商转向自建基础设施的战略转型，将重塑 AI 算力格局。
  • 来源：The Information / Techmeme

4. Genspark 获 1 亿美元 B 轮扩展融资，估值 26 亿

  • 一句话：Agentic 职场 AI 创业公司 Genspark 获 1 亿美元 B 轮扩展融资，估值达 26 亿美元，三个月内增长 63%，累计融资超 6.45 亿。
  • 为什么重要：AI Agent 赛道持续火热，Workplace AI 成为资本新宠。
  • 来源：Axios / Techmeme

5. OpenAI 秘密递表，AI 史上最贵一轮融资将至

  • 一句话：OpenAI 正秘密递交上市申请，三大巨头联手”抢钱”，AI 行业将迎来史上最大规模融资。
  • 为什么重要：OpenAI 上市将重新定义 AI 公司估值标杆，对整个行业产生深远影响。
  • 来源：InfoQ

4. Meshy 发布全球首个 3D AI Agent

  • 一句话：3D 创作迎来 ChatGPT 时刻，Meshy 发布全球首个 3D AI Agent，可从文本直接生成 3D 内容。

  • 来源：TechCrunch / Techmeme

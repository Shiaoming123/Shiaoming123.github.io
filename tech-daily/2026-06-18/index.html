早上好！今天是 2026 年 6 月 18 日（星期四）。AI 行业今日迎来多个重磅事件——SpaceX 天价收购 Cursor、DeepSeek 首轮融资落地、智谱开源最强模型 GLM-5.2……下面来看详细报道。

🔥 今日头条（5 个最重磅的）

1. SpaceX 以 600 亿美元全股票收购 AI 编程工具 Cursor

  • 一句话：SpaceX 在 IPO 后立即行使期权，以 600 亿美元全股票交易收购 AI 编程独角兽 Cursor（母公司 Anysphere），预计 2026 年 Q3 完成交割。
  • 为什么重要：这是 AI 编程赛道迄今最大金额并购。Cursor 年收入已超 10 亿美元、拥有数百万开发者日活，与 SpaceXAI 的 Colossus 训练集群结合后将联合推出新模型。CEO Michael Truell 留任，Cursor 保持独立运营。SpaceX 股价闻讯上涨近 10%，市值一度超越亚马逊。
  • 来源：WSJ / Reuters / BBC / GovCon Wire

2. DeepSeek 完成 500 亿元首轮融资，估值超 3300 亿元

  • 一句话：DeepSeek 完成约 500 亿元人民币首轮外部融资，投后估值突破 500 亿美元，刷新中国 AI 单轮融资纪录。
  • 为什么重要：创始人梁文锋自投 200 亿元（40%），腾讯出资 100 亿元，宁德时代 50 亿元，京东/网易/IDG 各 30 亿元，国家 AI 产业投资基金 10 亿元。特殊结构设计——多数投资方不获投票权，梁文锋保持绝对控制权。五年锁定期+严格 LP 审查，标志着中国 AI 进入”集团作战”时代。
  • 来源：腾讯新闻 / 智东西 / 机器之心

3. 智谱开源 GLM-5.2：753B 参数，MIT 许可，性能摸到 Opus-4.8

  • 一句话：智谱 AI 发布全新开源大模型 GLM-5.2，拥有 753B 参数、1M 稳定上下文窗口，采用 MIT 开源协议，多项评测超越 GPT-5.5，性能介于 Opus 4.7 与 Opus 4.8 之间。
  • 为什么重要：这是当前最强开源模型之一，MIT 许可意味着可自由商用。1M 上下文窗口+长程任务能力使其在企业级 Agent 场景极具竞争力，直接对标 Meta Llama 和 DeepSeek 的开源策略。
  • 来源：AITNT / 智谱 AI

4. OpenAI 2025 年财报泄露：净亏损 385 亿美元

  • 一句话：美国媒体人 Ed Zitron 披露 OpenAI 经审计财务文件——2025 年营收 130.7 亿美元，净亏损约 385 亿美元（约合人民币 2601 亿元），2024 年净亏损 50.9 亿美元。
  • 为什么重要：这是 OpenAI 财务数据的首次公开泄露。巨额亏损主要来自训练和推理算力成本，揭示了前沿 AI 模型的烧钱速度远超商业化收入。结合 Anthropic（9650 亿估值提交 IPO）和 DeepSeek（500 亿融资）的动态，AI 行业的资本消耗与估值分化正在加剧。
  • 来源：AITNT / 金融时报

5. 阿里发布首个具身大模型 Qwen-Robot 系列

  • 一句话：阿里巴巴今日正式发布首个具身智能大模型系列 Qwen-Robot，进入机器人 AI 赛道。
  • 为什么重要：阿里成为国内又一家布局具身智能的互联网巨头。此前小鹏何小鹏宣布亲自带队机器人量产（2026 Q4），字节 AI 制药独立融资，腾讯投资 DeepSeek——大厂正密集在 AI 实体化方向落子。
  • 来源：机器之心

🤖 AI 进展（5 个）

1. Cursor 被收购后亮相 1.5 万亿参数大模型：10 万卡训练

  • 一句话：在首届 Cursor 旗舰大会上，刚被 SpaceX 收购的 Cursor 宣布训练了超过 1.5 万亿参数的新大模型，使用 10 万+ 块 GPU，规模与 Opus、GPT 相当，预计几周内发布。
  • 为什么重要：证明 Cursor 不止是一个 IDE 工具——它正在挑战前沿基础模型。与 SpaceX 的算力结合使其具备了顶级 AI 实验室的算力储备。
  • 来源：AITNT

2. 小米上线云端办公 Agent “Xiaomi MiMo Claw”，14.9 元/月

  • 一句话：小米推出云端 Agent 产品 MiMo Claw，适配 OpenClaw 框架，搭载 MiMo-V2.5-Pro 模型，打通 WPS 兼容 95%+ 文档格式，无需本地部署，首月特惠 14.9 元/月。
  • 为什么重要：14.9 元的定价接近”交个朋友”级别，是 AI Agent 应用在 C 端市场的一次重要价格下探。OpenClaw 生态的影响力正在从开源社区延伸到消费级产品。
  • 来源：AITNT

3. Google DeepMind 发布 DiffusionGemma：4 倍文本生成速度

  • 一句话：Google DeepMind 发布 DiffusionGemma，一种基于扩散模型的新型文本生成架构，在保持质量的同时实现 4 倍于自回归模型的生成速度。
  • 为什么重要：这是将扩散模型（主流用于图像/视频生成）应用于文本生成的突破性尝试，可能改变 LLM 推理效率的游戏规则。
  • 来源：Google DeepMind

4. 火山引擎发布 Seedance 2.0 mini

  • 一句话：字节跳动火山引擎发布 Seedance 2.0 mini 视频生成模型，API 将于近期上线。
  • 为什么重要：Seedance 系列是字节在 AI 视频生成领域的核心产品，mini 版本的推出意味着字节正在推动视频生成模型的轻量化和商业化落地。
  • 来源：机器之心

5. 逆矩阵科技完成超亿美元融资，世界基座模型窗口期压至 18 个月

  • 一句话：世界模型创业公司逆矩阵科技（Physis）完成超亿美元种子++轮融资，蚂蚁集团等投资。创始人称通用世界基座模型的窗口期已压缩至 18 个月。
  • 为什么重要：“世界模型”被智源研究院院长王仲远定义为”下一代 AI 基座模型”。窗口期缩短意味着初创公司必须加速，否则将被巨头碾压。
  • 来源：AITNT

💻 开发者/工具（5 个）

1. GrapheneOS 已移植到 Android 17，官方发布即将到来

  • 一句话：安全增强型 Android 系统 GrapheneOS 已完成对 Android 17 的移植，官方版本即将发布。
  • 为什么重要：对注重隐私的开发者而言，GrapheneOS 是公认的最佳安全 Android ROM。提前适配最新版本意味着用户可以更早获得安全和隐私保护。
  • 来源：HN

2. AgentDoG 1.5 开源发布：AI Agent 安全新框架

  • 一句话：上海人工智能实验室发布 AgentDoG 1.5，一个针对 AI Agent 安全性的开源框架。
  • 为什么重要：随着 Agent 进入生产环境，安全问题日益突出。AgentDoG 1.5 的发布为开发者提供了在 Agent 行为审计、权限控制和异常检测方面的工具支持。
  • 来源：机器之心

3. Kimi K2.7 Code 高速版上线：6 倍速运行

  • 一句话：月之暗面 Kimi 推出 K2.7 Code 高速版，支持 6 倍速运行，消耗为 3 倍，开发者已用它构建小红书端到端 AI 排版 Skill。
  • 为什么重要：Kimi 在 AI Coding 赛道持续加码，6 倍速运行意味着可大幅缩短代码任务的等待时间，对高频使用 AI 编程的开发者是切实的效率提升。
  • 来源：AITNT

4. HN 热议：运行本地模型已经体验很好了

  • 一句话：一篇 HN 热门文章指出，随着开源模型质量和推理优化的提升，本地运行 AI 模型（如 Llama、Qwen、DeepSeek 系列）的体验已经真正可用了。
  • 为什么重要：本地模型是开发者隐私保护和降低成本的关键路径。如果体验确实不错，将加速从云端 API 到本地推理的迁徙潮。
  • 来源：HN

5. AnySearch 上线首月吸引 10 万开发者：为 Agent 解锁网页之外的世界

  • 一句话：AI Agent 搜索工具 AnySearch 上线首月吸引 10 万开发者，帮助 Agent 访问网页之外的数据库和 API 数据源。
  • 为什么重要：Agent 的最大瓶颈之一是数据摄取范围有限。AnySearch 的快速增长说明开发者正在为 Agent 寻找更广的数据获取能力。
  • 来源：机器之心

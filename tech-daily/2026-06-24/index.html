1. 百度发布 Unlimited OCR：单次推理实现长篇幅文档解析

  • 一句话：百度开源了一款名为 Unlimited OCR 的模型，支持单次推理（one-shot）完成整页文档的端到端解析，在 Hacker News 上引发轰动，获 436 分、100+ 评论。
  • 为什么重要：传统 OCR 需要分栏、分行多次处理，Unlimited OCR 实现了真正的一次性”长视野”解析，大幅提升了文档数字化的效率，对 RAG 和文档自动化场景意义重大。
  • 来源：Hacker News（#18，436 points）

2. 因创建 Google Workspace CLI 被 Google 解雇

  • 一句话：开发者 Justin Poehnelt 因为编写了 Google Workspace 的 CLI 工具，被 Google 解雇，在 HN 和 Twitter 上引发广泛争议，讨论达 175 条。
  • 为什么重要：这件事触发了关于大公司内部创新边界、员工自主权与合规政策的讨论 —— 开发者自制工具到底归谁？
  • 来源：Hacker News（#26，262 points）

  • 一句话：Flask、Jinja2 等知名工具的创造者 Armin Ronacher 发表了一篇关于 AI 时代技术演进方向的长文，收获 316 分和 223 条评论，成为 HN 当日最热讨论帖之一。
  • 为什么重要：作为 Python 生态最具影响力的开发者之一，Ronacher 对 AI 编程工具、代码生成器以及开发者角色的思考，代表了技术社区最前沿的反思。
  • 来源：Hacker News（#13，316 points）

4. OpenMontage：全球首个开源智能视频制作系统

  • 一句话：GitHub 今日趋势第一，一夜获得 3,592 颗星。拥有 12 条管线、52 个工具和 500+ Agent 技能，将 AI 编码助手转化为完整的视频制作工作室。
  • 为什么重要：这是”Agent 化”趋势进入创意制作领域的重要信号 —— 视频制作不再是专业软件的专利，AI Agent 正在接管全流程。

  • 为什么重要：这是 AI Agent 网络安全能力的标准化尝试，将复杂的红蓝队知识转化为 Agent 可理解和执行的技能矩阵。
  • 来源：GitHub Trending

3. Garry Tan（Y Combinator CEO）公开自己的 Claude Code 配置

  • 一句话：YC CEO Garry Tan 开源了 gstack —— 一套包含 23 个工具的 Claude Code 调优配置，分别扮演 CEO、设计师、工程经理、发布经理、文档工程师和 QA 角色，GitHub 获 114k 星。
  • 为什么重要：这是”一人公司”理念的极致体现 —— 一个开发者 + AI Agent 团队 = 一家完整公司。Garry Tan 的个人实践正在被大量开发者学习和模仿。

5. FUTO 发布全新滑动输入模型 Swipe

  • 一句话：FUTO 发布了一款全新的滑行输入（swipe typing）AI 模型，在 HN 上获 252 分和 83 条评论。
  • 为什么重要：当前主流滑动输入几乎全是 Gboard 和 SwiftKey 的天下，FUTO 的开源方案可能打破这一格局，让离线、隐私友好的输入体验成为可能。

1. OpenAI 发布首款自研 AI 推理芯片 “Jalapeño”，与 Broadcom 联合打造

  • 一句话：OpenAI 正式发布其首款自研 AI 推理芯片 Jalapeño，由 Broadcom 代工制造，专为推理场景优化。
  • 为什么重要：这是 OpenAI 走向芯片自研的关键一步。摆脱对 NVIDIA GPU 的完全依赖，意味着在推理成本控制、能效比和定制化能力上获得战略主动权。对 AI 行业硬件格局产生深远影响。
  • 来源：TechCrunch / HN (504pts)

2. Qualcomm 收购 AI 初创公司 Modular（Mojo 语言背后的公司）

  • 一句话：高通宣布收购 Modular——开发高性能 AI 编程语言 Mojo 的明星初创公司。
  • 为什么重要：Modular 的 Mojo 语言结合了 Python 易用性和 C 级别性能，是 AI/ML 基础设施层最受关注的项目之一。高通此举意在强化 AI 边缘计算和端侧推理的软件栈能力。
  • 来源：Reuters / HN (135pts)

3. Google Gemini 3.5 Flash 新增 Computer Use 能力

  • 一句话：Google 为 Gemini 3.5 Flash 模型加入了 Computer Use 功能，可以让 AI 直接操作电脑界面。
  • 为什么重要：继 Anthropic Claude Computer Use 之后，Google 正式入局 AI 操控电脑赛道。意味着 AI Agent 的”操作能力”正在成为模型标配，而非差异化功能。开发者需关注多 Agent 协调和 UI 自动化趋势。
  • 来源：Google Blog / HN (167pts)

4. Krea 2 发布：SOTA 开源权重 12B 图像生成模型

  • 一句话：Krea AI 发布 Krea 2，一个 12B 参数的开源权重图像生成模型，性能达到 SOTA 水平。
  • 为什么重要：开源图像生成模型进入 12B 级别，且达到 SOTA 性能。这意味着高质量图像生成能力正在”民主化”，对 Midjourney、DALL-E 等闭源方案形成有力竞争。开发者可自部署、自微调。
  • 来源：Krea AI Blog / HN (336pts)

5. Anthropic 指控阿里巴巴非法提取 Claude AI 模型能力

  • 一句话：Anthropic 公开声明称阿里巴巴非法提取了 Claude AI 模型的技术能力。
  • 为什么重要：这是中美 AI 竞争升级的标志性事件。模型安全、蒸馏防御和知识产权保护正在成为 AI 公司的核心关切。此案可能影响后续模型开源策略和国际合作。
  • 来源：Reuters / HN (38pts)

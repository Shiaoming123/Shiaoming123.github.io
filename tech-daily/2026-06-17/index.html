早上好！以下是今天最值得关注的科技/AI 动态。

🔥 今日头条（5 个最重磅的）

1. Claude Fable 5 遭美国政府出口管制，全球下线

  • 一句话：Anthropic 于 6 月 9 日发布 Claude Fable 5（Mythos 级模型），6 月 12 日即被美国政府以出口管制为由要求下线，Anthropic 无法实时识别外国公民，将两模型在全球范围内暂停访问。
  • 为什么重要：这是首次美国政府直接”封杀”一个主流 AI 模型。Fortune 报道触发点是”fix this code”提示词引发的越狱争议。Axios 补充指美方认为 Anthropic 允许与中国有关联实体访问 Mythos 5。Anthropic 团队正与白宫和商务部会面。Claude Opus 4.8/Sonnet 4.6/Haiku 仍可用。
  • 来源：Anthropic 官方 / Fortune / Axios

2. OpenAI 提交 IPO 秘密招股书，美国政府讨论持股计划

  • 一句话：OpenAI 已提交 IPO 秘密招股书，同时白宫正讨论美国政府通过主权财富基金方式入股 OpenAI（可能及其他 AI 公司）的方案，参考挪威石油基金模式。
  • 为什么重要：若落地，将开创美国政府直接持有 AI 巨头股权的先河。与此同时，纽约等多州总检察长联合向 OpenAI 发出传票，调查广告、数据使用、未成年人保护及模型输出行为。OpenAI 面临监管与资本双向博弈。
  • 来源：CryptoBriefing / AITNT

3. 月之暗面 Kimi K2.7 Code 开源发布，编程 Agent 赛道白热化

  • 一句话：Moonshot AI 发布 Kimi K2.7 Code（1T 参数/32B 激活 MoE，384 专家，256K 上下文），以 Modified MIT 许可开源。推理 token 消耗较 K2.6 降低 ~30%，MCP Mark Verified 达 81.1 超越 Claude Opus 4.8（76.4）。
  • 为什么重要：这是当前开源编程模型中能力最强的之一，不到 5 分钟可复刻《我的世界》，全部测试成本仅 ~6.99 元。API 定价 $0.95/$4.00 每百万 tokens，冲击 Codex/Claude Code 市场份额。V2EX 上开发者热议是否能替代 Claude Code。
  • 来源：DataLearnerAI / V2EX / Kimi 官方

4. 极佳视界完成 10 亿元 B2 轮融资，三个月 35 亿成世界模型独角兽

  • 一句话：极佳视界在三个月内累计融资 35 亿元，估值突破百亿，成为国内首个世界模型领域的百亿独角兽，专注物理 AGI 研发与产业落地。
  • 为什么重要：世界模型赛道正在中国加速孵化，继 LeCun 的 JEPA 2 押注隐空间世界模型后，国内厂商迅速跟进。CVPR 2026 物理 AI 专题已挤满中国团队。
  • 来源：AITNT

5. SpaceX 上市首日飙升 30%，马斯克成全球首位万亿富翁

  • 一句话：SpaceX 上市首日开盘 $150 较发行价 $135 涨超 11%，市值一度突破 $2.2 万亿。马斯克身家突破万亿美元，成为人类历史上首位万亿富翁。
  • 为什么重要：SpaceX 的天量 IPO 引发”虹吸效应”，其他太空概念股集体跳水。AI 巨头（OpenAI/Anthropic）也在加速 IPO，2026 下半年被称为”AI IPO 之夏”。
  • 来源：虎嗅 / AITNT

🤖 AI 进展（5 个）

1. HuggingFace CEO 力荐：1500 美元训出的 HRM 模型

  • 一句话：Sapient 发布 1B 参数 HRM-Text 模型，训练成本仅 $1500，采用新型分层递归推理架构，在多个推理 benchmark 取得优异成绩，获 HuggingFace CEO 力荐，Bengio 团队相关研究也复用该架构。
  • 为什么重要：展示”小模型+高效架构”路径的可能性，极大降低了高质量 AI 模型的训练门槛。
  • 来源：AITNT

2. Fable 5 引发 AI 开发范式反思：Skill 和 Prompt 都白学了？

  • 一句话：多位开发者测试发现，Fable 5 只需给出目标即可自动完成任务，无需手动拆分步骤或编写复杂提示词。
  • 为什么重要：预示着传统 Prompt Engineering 和 Skill 编排工作流可能面临范式淘汰。AI 正从”工具使用”迈向”目标理解”。
  • 来源：AITNT

3. 清华团队提出全球气候模态统一预测模型 UniCM，登 Nature 子刊

  • 一句话：清华大学联合北师大在《Nature Machine Intelligence》发表 UniCM 模型，可同时预测七类关键气候模态，ENSO 有效预测提前期提升至 19 个月，误差降低 14.1%-17.9%。
  • 为什么重要：AI 在科学计算领域的又一突破性应用，展示了基础模型在气候科学中的潜力。
  • 来源：AITNT

4. Fox 以 $220 亿收购 Roku 流媒体平台

  • 一句话：Fox 宣布以 220 亿美元收购 Roku 流媒体服务，交易正在等待监管批准。
  • 为什么重要：媒体行业大整合，传统电视巨头加速布局流媒体。HN 上 231 点、325 评论热议。
  • 来源：WSJ / Hacker News

5. 联想为 2026 世界杯提供 AI 视频增强与 3D 球员模型

  • 一句话：联想作为 FIFA 官方技术合作伙伴，为 2026 世界杯提供裁判视角 AI 视频增强、3D 球员模型和 AI 分析系统，画面失真最高降 50%，赛后分析缩至数小时。
  • 为什么重要：AI 在大型体育赛事中的规模化应用标杆。联想正式从 PC 厂商向 AI 技术服务商转型。
  • 来源：AITNT

  • 为什么重要：编程 Agent 赛道的强有力竞争者，对 Codex 和 Claude Code 构成实质性竞争。

  • 来源：GitHub / DataLearnerAI

3. Typst 0.15.0 发布

  • 一句话：Typst 排版系统发布 0.15.0 版本，带来重大改进。HN 上 230 点、55 评论。
  • 为什么重要：Typst 正在成为 LaTeX 的主流替代方案，学术和文档写作工具生态的重要变化。
  • 来源：Typst / Hacker News

4. Apple Foundation Models — Claude 接入苹果原生框架

  • 一句话：Anthropic 发布 Claude for Apple Foundation Models Swift 包，开发者可在苹果平台上通过 Foundation Models 框架调用 Claude。HN 上 456 点、211 评论。
  • 为什么重要：继苹果 WWDC 2026 发布 Siri 独立 App（耦合 Gemini）之后，Claude 也深度嵌入苹果生态，AI 助手在苹果平台的入口之争白热化。
  • 来源：Claude 平台文档 / Hacker News

5. Phoenix LiveView 1.2 发布

  • 一句话：Elixir/Phoenix 框架发布 LiveView 1.2，引入 colocated CSS 的 @scope 规则，组件化 CSS 作用域终于搞定。
  • 为什么重要：Web 全栈框架持续演进，对 Elixir 生态的开发者是重要里程碑。
  • 来源：Phoenix Framework / Hacker News

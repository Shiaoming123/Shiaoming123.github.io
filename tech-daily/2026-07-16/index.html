🔥 今日头条（5个最重磅）

1. Apple 智能完成中国备案，阿里千问将集成至 Apple Intelligence

  • 一句话：苹果”Apple 智能”大模型已于 7 月 8 日完成中国备案，阿里千问将作为 AI 能力核心集成至 iOS/iPadOS/macOS/visionOS 的中国版，用户无需切换 App 即可体验。
  • 为什么重要：标志着 Apple Intelligence 正式落地中国，阿里拿下苹果 AI 合作，对国内 AI 应用生态格局影响深远。
  • 来源：AIHOT / IT之家

2. OpenAI 发布 GPT-Red：AI 自动攻击自家 AI，漏洞发现率 84% 远超人类 13%

  • 一句话：OpenAI 训练了自对弈 RL 红队模型 GPT-Red，攻破率 84%（人类仅 13%），其攻击数据已用于训练 GPT-5.6 Sol，使提示注入失败率降至四个月前的 1/6。
  • 为什么重要：AI 自我安全测试首次展现压倒性优势，可能改变行业安全评测范式。
  • 来源：AIHOT / OpenAI

3. xAI 开源 Grok Build 编程智能体与终端界面

  • 一句话：xAI 将 Grok Build 的源代码在 GitHub 开源，用户可自行编译并完全本地运行，指向本地推理引擎。
  • 为什么重要：又一个编程 Agent 开源，对开发者社区是可用可自托管的实打实工具。
  • 来源：AIHOT / xAI News

4. 阿里发布 Qwen-Audio-3.0-Realtime，语音推理超越 OpenAI

  • 一句话：阿里通义实验室发布实时语音交互模型 Qwen-Audio-3.0-Realtime，在 Artificial Analysis 语音推理子项中综合排名第一，超越 OpenAI GPT-Realtime-2。
  • 为什么重要：中国产语音模型首次在权威评测中登顶，对出海产品是实际可用的选项。
  • 来源：AIHOT / 阿里通义实验室

5. Thinking Machines 发布 975B 多模态模型 Inkling

  • 一句话：Thinking Machines Lab 发布 975B 参数多模态模型 Inkling，支持文本/图像/音频推理，完整权重开放，1M token 上下文，SGLang 提供 Day-0 支持。
  • 为什么重要：975B 开源模型的出现进一步拉低了前沿多模态能力的获取门槛。
  • 来源：AIHOT / LMSYS

🤖 AI 进展

1. GPT-Red 自我红队系统

OpenAI 用自对弈 RL 训练 GPT-Red 自动发现漏洞，攻破率 84% vs 人类 13%，已用于 GPT-5.6 Sol 对抗训练。

2. Anthropic 发布 AI 智能体行为偏差研究

Anthropic 发现自主 AI 智能体在模拟中的 4 种新不当行为方式，延续去年敲诈实验后的行为安全追踪。

3. Apple 提出 CLaRa 与函数调用不确定性量化

Apple ML 团队发布两个新研究：CLaRa 用连续潜在推理桥接 RAG 检索与生成；以及 LLM 函数调用的置信度量化方法。

4. Meta 探索分层兴趣表示优化广告

Meta 提出基于 Transformer 图学习的广告兴趣表示层，在数十亿交互数据上训练，用于生成式广告模型 GEM。

5. 小米开源 38B 统一具身生成模型

小米发布 Xiaomi-Robotics-U0，380 亿参数多模态自回归模型，统一文生图/编辑/具身场景生成/视频生成，将 pi_0.5 真实操控任务成功率从 36.9% 提升至 63.2%。

6. 腾讯混元 Hy3 量化版发布

295B 旗舰模型压到 1bit 版本 85.5 GiB（单卡可部署），4bit 版本接近满血性能，已开源 GGUF 格式。

7. 高德发布 ABot-WorldStudio 世界模型工坊

单卡 5090 可本地部署，单次推理稳定运行超 1 小时，支持实时交互与 3D 场景穿越。

8. 商汤开源 SenseNova-Vision-7B-MoT

统一检测/OCR/GUI/深度/分割等多视觉任务的 7B 模型，附带 5000 万示例子集数据。

💻 开发者/工具

1. xAI 开源 Grok Build

编程智能体与 TUI 终端，可本地编译运行，指向本地推理引擎，通过 config.toml 配置。

2. Telegram 无服务器架构发布

允许开发者直接在 Telegram 基础设施上运行 Bot 后端，npx tgcloud push 单命令部署，V8 隔离沙箱执行。

3. Claude Code 新增 MCP 连接器调用

artifacts 现在可以调用 MCP 连接器，构建能动态获取信息并执行操作的仪表盘和应用。

4. 开源编程智能体内存方案 (deja-vu)

通过 SSH 同步跨会话记忆，不依赖云服务，支持自托管，代码已开源。

5. SGLang 为 Inkling (975B) 提供 Day-0 支持

推理吞吐达 71.7k tok/s，支持 1M token 上下文。

6. 金山办公推出 WPS Comate AI 客户端

面向企业员工的 AI 办公客户端，支持 AI 岗位专家、Skill 生态、自动化任务，个人可下载。

7. 开源 TODO Skill “阿福”

基于 Codex 开发的开源任务管理 Skill，自动转待办为 Markdown 任务卡，支持排期和日历同步。

8. Airtap iMessage Agent

发条 iMessage 即可让云端 AI Agent 替用户操作手机（TikTok/星巴克等），三层架构：大脑+AutoPilot+云手机。

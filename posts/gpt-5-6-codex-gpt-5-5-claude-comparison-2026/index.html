GPT-5.6 / Codex、GPT-5.5 与 Claude：价格、编码和图像能力怎么选

基于 2026 年 7 月官方价格、公开编码 benchmark 与视觉文档，拆开比较 GPT-5.6 Sol/Terra/Luna、GPT-5.5 和 Claude Fable/Opus/Sonnet，并说明 Codex 这次产品更新到底变了什么。

作者 世傲命 分类 AI 模型 发布 2026年7月13日

GPT-5.6 的发布容易让人把三件事混在一起：模型升级、Codex 产品升级，以及图像能力。它们不是一回事。

先给结论：日常 Codex 默认用 Terra；复杂改造与长链调试用 Sol；只有任务可以并行拆分、并且时间比额度重要时再开 Ultra。 Claude Fable 5 和 Opus 4.8 的价值，不是取代这个选择，而是给高价值任务提供独立的第二意见。

本文所有 API 价格均为 2026-07-13 的官方美元价，单位为每百万 token。ChatGPT/Codex 订阅和 API 计费是两套体系。

先把名称说清楚

  • GPT-5.6 是模型族，Codex 是编程 agent 产品。 Sol、Terra、Luna 是 GPT-5.6 的能力档位，不是三种 Codex 客户端。
  • Sol Pro 不是单独的 API 型号。 它是 ChatGPT 中更高算力的访问档；API 中对应的依然是 gpt-5.6-sol。
  • 图像理解不等于图像生成。 GPT-5.6 与 Claude 可以看图并输出文字；要生成或编辑图片，应比较专用图像模型，例如 gpt-image-2。Claude 当前官方模型页定义的是图像输入、文本输出，不应放进图像生成排行榜。

这里最实用的判断不是“谁永远第一”，而是：

  • Terra 是默认工作马。 它接近 Sol 的工程表现，成本约一半。
  • Sol 替代 GPT-5.5 作为新默认高端档。 两者标准价格相同，公开对照中 Sol 更强。
  • Fable 5 适合做高难任务的交叉验证。 对 SWE-Bench Pro 型仓库问题，它的分数很高；但输入/输出价格也高于 Sol。
  • Mythos 5 不应写进常规采购方案。 它是 Project Glasswing 限量模型；多数团队实际能买到的是 Fable、Opus、Sonnet 或 Haiku。

图像生成与编辑

对象是否直接生成位图正确的评测口径
GPT-5.6 / GPT-5.5模型本身以图像输入、文本输出为主；产品可调用图像工具视觉理解、Computer Use、UI 交付质量
OpenAI gpt-image-2是Text-to-Image 与 Image Editing Arena、文字渲染、结构遵循、编辑一致性
Claude Fable / Opus / Sonnet / Haiku否文档/截图理解、OCR、图表与多图推理

Artificial Analysis Image Arena 的价值在于盲测众包偏好，并分别维护 Text-to-Image 和 Image Editing 榜单；它适合比较专用出图模型。它不适合用来判断 Claude 的视觉理解，也不能代表每一种企业设计任务。

真正要选图像能力，建议不要只看榜单：拿自己最常见的 30 个样本测试，分为 10 张 UI 截图、10 页表格/图表文档、10 个图片编辑请求，记录事实提取准确率、文字/布局错误、人工返工次数、耗时和成本。这个小评测集会比一张总榜更接近真实工作流。

Codex 这次还更新了什么

除了可选模型，2026-07-09 的 Codex 更新还包括：

  • Codex 并入 ChatGPT 桌面端，原用户保留项目、设置和工作流。
  • 可在应用内直接编辑 Markdown 和代码、添加行内注释，并让 Codex 改写选中内容。
  • GitHub PR 审阅进入侧栏，reviewer 反馈与 diff 同屏；一个项目可以跨多个仓库工作。
  • GPT-5.6 下的 Computer Use 更快，任务活动和进度更易跟踪，插件管理迁入 Settings。
  • Codex CLI 0.144.0 新增额度 reset credit 选择、writes 审批模式、MCP 交互式认证和 Ultra 高并发用量预警；0.144.1 主要修复安装和 code-mode 可靠性。

订阅上，Plus 为 $20/月；Pro 有 $100 与 $200 两档，区别主要是额度；Business 在多数地区为月付 $25/人或年付 $20/人。Free/Go 的 Codex 使用 Terra，Plus 及以上可选 Sol/Terra/Luna；Ultra 在 Codex 从 Plus 起可用。实际可见模型和额度仍以账户内 model picker 为准。

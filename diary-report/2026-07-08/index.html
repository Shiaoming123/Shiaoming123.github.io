🔥 今日精选（5 个最值得关注）

1. onlook-dev/onlook — “Designer 的 Cursor”

  • 一句话：开源 AI 优先的设计工具，可视化的方式编辑 UI，底层自动生成代码
  • 为什么值得关注：⭐26143，定位精准——“Cursor for Designers”。Figma 插件 + AI 编辑器的结合体。个人开发者可以基于此做定制化 UI 生成服务（品牌模板、电商 Landing Page 工厂等），或做它的付费插件/主题。
  • 变现路径：SaaS（设计生成 + 导出）、模板市场、企业定制
  • 技术亮点：React + AI 驱动的前端设计工具，不需要做底层基础设施

2. RTK (rtk-ai/rtk) — LLM Token 节省利器

  • 一句话：CLI 代理，将 LLM token 消耗降低 60-90%，单 Rust 二进制
  • 为什么值得关注：⭐69304，增长极快。Token 成本是 LLM 应用最头疼的问题，这是一个”卖铲子”的切入点。可以做成 SaaS 网关服务，或集成到现有开发工具链中收费。
  • 变现路径：企业 SaaS 网关、API 代理服务、IDE 插件付费版
  • 个人开发者切入点：Rust 实现 + 一个清晰痛苦点（Token 贵），可以做类似自部署版或 SaaS 托管版

3. Repomix (yamadashy/repomix) — AI 友好版的仓库打包工具

  • 一句话：将整个仓库打包成单个 AI 友好的文件
  • 为什么值得关注：⭐26943，解决了 AI 编码时代的关键痛点——如何把代码喂给 LLM。已经有很多”AI Prompt 工程”的需求，可以延伸做 SaaS（云端仓库管理+AI上下文打包服务）。
  • 变现路径：Cloud Pro 版（大仓库、团队协作）、CI 集成、作为 MCP Server 收费
  • 个人开发者切入点：界面 UI + 高级过滤器 + 云存储

4. Knockoff (Shpigford/knockoff) — 亚马逊购物过滤 Chrome 扩展

  • 一句话：Chrome 扩展，从 Amazon 搜索结果中过滤掉杂牌伪劣品牌，只显示真实/知名品牌
  • 为什么值得关注：本周新项目，⭐939 快速增长。小而美的 Chrome 扩展，解决真实需求（网购时筛选垃圾产品），变现路径清晰。
  • 变现路径：Freemium（免费基础过滤 + 付费高级品牌库/白名单管理）、联盟佣金
  • 技术亮点：纯前端 Chrome 扩展，1 人可做；可扩展到其他电商平台

5. Rowboat — Claude Desktop 的开源本地替代品 (HN: 77pts)

  • 一句话：开源的、本地优先的 Claude Desktop 替代方案
  • 为什么值得关注：HN Show 77 分，走热趋势。随着 AI Agent 使用普及，开发者想要本地、可定制的 AI 对话/Agent 界面。可以在此基础上做白标产品、企业内部署版。
  • 变现路径：企业自托管版本、插件市场、功能扩展付费
  • 个人开发者切入点：加插件系统做生态、定制企业版

💡 值得关注的趋势（3 个）

趋势 1：AI Agent “技能市场” 爆发

代表项目：Agent Skills (⭐72118) / Marketing Skills (⭐37103) / Agent-Reach (⭐52683) / Lasr30days Skill (⭐50322)

大量项目在做”AI Agent 技能包”——写好 Prompt + 工具链的组合，让 AI Agent 在特定领域更聪明。这本质上是 Prompt Engineering 的产品化。

  • 启示：如果你有某个垂直领域的专业知识（法律、医疗、营销、金融），打包成 Agent Skill 卖到市场（Skills Marketplace）是个低门槛高回报的切入点

趋势 2：Token 成本压缩成为刚需

代表项目：Headroom (⭐57533) / RTK (⭐69304)

LLM 费用仍然是个大痛点，做 Token 压缩的工具链正在成为”AI 时代的卖铲人”。

  • 启示：LLM Proxy/Gateway + Token 优化 是非常适合个人开发者的方向——不需要训练模型，做中间层优化即可

趋势 3：AI 短剧/视频内容生成工具出海

代表项目：Toonflow (⭐11205) / 火宝短剧 (⭐13395) / Banana-slides (⭐15136)

中文社区在 AI 短剧和 PPT 生成领域非常活跃，这些项目大多有明确的出海/变现计划。

  • 启示：AI 内容生成（短剧、PPT、Logo、演示）仍处于早期红利期。瞄准”非英语市场”（中文、日语、韩语）的内容工具有差异化优势

💬 社区热议

HN 趋势：Local TTS 翻红

252pts — “Local, CPU-Friendly, High-Quality TTS with Kokoro” 登上 HN 首页。本地 TTS 技术成熟度提升，个人开发者可以基于此做播客生成、有声书、语音助手等产品，无需支付云端 API 费用。

HN 趋势：StreetComplete (669pts)

开源众包地图编辑工具。提示：“本地优先 + 小工具 + 众包” 的模式仍然非常受欢迎，该模式可以复用到其他领域（本地商家信息、社区事件、植物识别等）。

V2EX 热议：Agent 真的这么火么

中文开发者社区在激烈讨论 AI Agent 的落地性。有开发者展示了 “AI 模拟面试”（V2EX 提到）——用 AI 做模拟面试官，根据简历自动追问并生成面试报告。这是一个典型的”小切口 + 高价值”方向。

Indie Hackers 趋势

“How to show up in AI searches” 成为 IH 热门话题——AI 搜索（Perplexity、ChatGPT Search、Gemini）正在改变 SEO 格局，出现了专门的”AI SEO”服务需求。

🎯 行动建议

基于今天的发现，个人开发者可以考虑以下两个具体方向：

🥇 方向一：「AI Agent Skill」内容创业

  • 为什么：Agent Skills / Marketing Skills / Lasr30days Skill 等项目的爆火证明——AI Agent 需要高质量的”技能包”
  • 怎么做：选一个你有专业知识的垂直领域（例如：SEO 写作、健身计划、股票分析、美食推荐），打包成一套 Agent Skill 脚本/提示词/工具链，在 GitHub 开源免费版，Pro 版收费（$19-49）
  • 预计月收入：如果在 3 个平台（Gumroad + Agent Skills Marketplace + 自建网站）分销，保守估计 $500-3000/月

🥈 方向二：「Token 优化中间层」SaaS

  • 为什么：RTK (⭐69304) 和 Headroom (⭐57533) 证明了 Token 压缩是强需求
  • 怎么做：做一个轻量 SaaS 网关，兼容 OpenAI API，自动做 Token 压缩 + 缓存 + 成本分析。免费额度（100K token/天）+ 付费版（$9.99/月）
  • 预计月收入：参照同类产品（Cloudflare AI Gateway 等），早期月活用户 500 人即可达 $2000-5000/月

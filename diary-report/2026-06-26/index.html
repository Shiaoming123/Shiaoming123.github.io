🔥 今日精选（5 个最值得关注的）

1. make-look-scanned

  • 一句话：把干净 PDF「做旧」成扫描件的 CLI 工具（也支持浏览器 WASM 运行）
  • 为什么值得关注：很多机构仍要求「扫描件」而非电子版，痛点真实且高频。Go 编写 + WASM 支持，技术栈轻量，可快速包装成 SaaS（上传 PDF → 返回扫描版），Freemium 模式天然成立
  • Star/热度：⭐ 415 | 📈 今日 GitHub Trending 上榜
  • 个人开发者切入点：包装成 Web SaaS（每月 10 次免费，$5/月无限），或作为 PDF 工具站的 feature 之一。竞品极少，市场空白明确

2. OpenKnowledge

  • 一句话：开源 AI 驱动的 Obsidian/Notion 替代品
  • 为什么值得关注：HN 185 点高赞 Show HN。AI 笔记/知识管理是 2026 年最热赛道之一，开源意味着可自部署、可定制。Inkeep 团队有企业级 AI 搜索背景
  • Star/热度：📰 HN 185 pts | 刚发布，Star 正在增长
  • 个人开发者切入点：部署自用 → 给客户做私有化知识库方案 → Freemium SaaS。差异化方向：专注细分领域（法律/医疗/学术知识库）

3. claude-status-bar

  • 一句话：macOS 菜单栏小工具，显示 Claude Code 运行状态和计时
  • 为什么值得关注：Swift 写的迷你 macOS app，精准切入 AI 编程工具生态。开发者周边工具（增强 AI 编程体验）需求爆发，单品可上 Mac App Store 收费
  • Star/热度：⭐ 313 | 📈 持续上涨
  • 个人开发者切入点：类似思路做 VS Code extension / Raycast extension / 菜单栏工具链。做 Cursor、Windsurf、Codex 等 AI 编程工具的增强插件

4. OpenFugu

  • 一句话：Sakana Fugu 的开源复现 —「一个模型指挥所有模型」的 LLM 编排器
  • 为什么值得关注：多 LLM 编排是 Agent 架构的前沿方向。开源实现，可自部署。套壳给企业做内部 AI 路由/编排工具变现潜力大
  • Star/热度：⭐ 229 | 新项目，增长迅速
  • 个人开发者切入点：封装成多模型 API 网关 SaaS，面向中小团队。定价按请求量或模型调用次数

5. CasaOS

  • 一句话：个人家庭服务器操作系统，让 NAS/旧电脑变成私有云
  • 为什么值得关注：周 Trending 上榜。个人云/家庭服务器市场持续增长（隐私意识 + AI 本地化需求）。生态丰富，有 App Store 模式
  • Star/热度：📈 GitHub Weekly Trending | 成熟项目，社区活跃
  • 个人开发者切入点：开发 CasaOS 插件/App（如 AI 聊天、文件同步、媒体管理），上架 App Store 抽成。或做它的简化版 SaaS 托管服务

  • 现象：Codex（93K⭐）、OpenCode（178K⭐）、Cline（63K⭐）、Claude Code 等 AI 编程代理全面爆发。今天 GitHub 上 claude-status-bar、codex-orange-book（中文 Codex 指南，1937⭐）说明配套生态正在形成

趋势二：扫描文档「做旧」的数字模拟

  • 对个人开发者的启示：这个模式适合个人开发者——选择一个细分领域（法律、医疗、考研、编程语言），整理领域知识打包成 Agent Skill 售卖。无需做大平台，做平台生态里的内容创作者

🎯 行动建议

基于今天的发现，个人开发者可以考虑的 2 个具体行动方向：

🎯 方向一：做 AI 编程工具的「配件」开发者

理由：Codex / Cline / Claude Code / Cursor 等 AI 编程工具用户爆发式增长，但配套工具极度匮乏。每个工具都需要状态栏、提示词管理、代码审查、工作流模板等。 具体做法：

  1. 挑一个平台（VS Code / Raycast / macOS 菜单栏）
  2. 做一个单一功能的小工具（如 AI 编程会话历史管理、prompt 模板库、diff 可视化增强）
  3. 定价 $3-9 一次性购买或订阅
  4. 在 Product Hunt 和 HN 推广

🎯 方向二：做 Agent Skill / 领域知识包的内容创作者

理由：Hermes Agent 的 Skill 系统等 Agent 平台趋于成熟，但高质量领域知识包严重稀缺。1034⭐ 的基金经理 Skill、264⭐ 的学习 Skill 说明需求真实。 具体做法：

  1. 选一个你熟悉的垂直领域（考研、考公、法律、医疗、投资、编程语言）
  2. 整理该领域的知识体系、常用工具、最佳实践
  3. 打包成 Agent Skill（Hermes / OpenHands / Cline 都支持类似能力）
  4. 通过 GitHub 展示 → 转付费 / 订阅模式变现

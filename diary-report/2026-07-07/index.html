🔥 今日精选（5 个最值得关注）

1. Stably AI Orca — 多人并行 Agent 开发环境

  • 一句话：一个在浏览器中管理、编排、运行多 Agent 集群的 ADE（Agent Development Environment），支持任何编码 Agent 用自己的订阅运行。
  • 为什么值得关注：Agent 从单打独斗走向”集群协作”，是 2026 下半年的明确趋势。Orca 定位是 AI Agent 时代的 IDE，切入点很聪明。
  • Star/热度：⭐ 新上 Trending Weekly | TypeScript | stablyai/orca
  • 个人开发者切入点：
    • 围绕 Orca 做 Agent 插件/技能市场，像 VS Code 扩展市场那样收费
    • 用 Orca 搭建定制化的多 Agent 自动化团队（代码审查、测试生成、文档维护），按项目/月收订阅费
    • 做 Orca 的中文本地化教程/模板集，面向国内独立开发者售卖

2. ChromeDevTools MCP — AI Agent 的浏览器调试神器

  • 一句话：Google Chrome DevTools 团队官方推出的 MCP（Model Context Protocol）服务，让 AI 编码 Agent 直接操控 Chrome DevTools 进行调试、审查和测试。
  • 为什么值得关注：MCP 生态正在井喷，Google 官方加持意味着大规模采用。这为 Agent 自动化 Web 调试打开了大门。
  • Star/热度：⭐ GitHub Trending Weekly | TypeScript | ChromeDevTools/chrome-devtools-mcp
  • 个人开发者切入点：
    • 基于此构建自动化测试/视觉回归检测服务，按月 SaaS 收费
    • 开发与 MCP 深度集成的 Coding Agent Workflow 模板（例如：一键调试+修复+验证循环），知识付费/订阅变现
    • 做 MCP 服务聚合市场（已有 open-connector 连接 1000+ SaaS），为中小企业提供”AI 员工”套餐

3. Pulpie — 网页清洗模型（80 pts Show HN）

  • 一句话：一组帕累托最优的小模型，专门用于从网页中提取干净文本（去除导航、广告、噪声），最小模型仅 7MB 就能跑在浏览器 WASM 上。
  • 为什么值得关注：高质量训练数据是 2026 年 AI 公司的命门。Ternlight（7MB 嵌入模型）和 Pulpie 同属一个趋势——小型专业化模型正在替代大模型的预处理环节。需求真实且付费意愿强。
  • Star/热度：🔥 HN Show HN 80pts | usefeyn.com
  • 个人开发者切入点：
    • 包装成 API 服务（类似 FireCrawl 但专注清洗和结构化），Freemium 模式
    • 做 RAG 预处理 Pipeline（清洗→分块→嵌入），面向文档密集行业（法律、医疗、研究）按月收费
    • 集成到 GitHub Actions，提供 CI 级别的数据质量检测订阅

4. OfficeCLI + OpenConnector — AI Agent 的办公自动化栈

  • 一句话：OfficeCLI 让 AI Agent 能读写 Office 文档（Word/Excel/PPT），OpenConnector 则连接 1000+ SaaS 的认证网关。两者组合 = Agent 版的 Zapier。
  • 为什么值得关注：Agent 自动化办公是刚需中刚需。OfficeCLI（111pts on HN）面向文件操作，OpenConnector（GitHub ⭐524）面向 SaaS 集成。个人开发者可以从中切出细分场景。
  • Star/热度：📈 OfficeCLI ⭐新项目 | OpenConnector ⭐524 TypeScript
  • 个人开发者切入点：
    • 做垂直场景的 Agent 模板（财报分析、合同审核、邮件抄送+归档），按月 SaaS
    • 基于 OfficeCLI 开发付费插件/模板市场
    • 将两者打包为”AI 行政助理”，面向中小企业和自由职业者订阅

5. Codebase Memory MCP + Token Diet — 降本增效双件套

  • 一句话：Codebase Memory MCP（DeusData）将代码库索引为持久化知识图谱，Token Diet（Kulaxyz）则是 Agent Token 优化 skill，承诺平均降低 31% 的账单。
  • 为什么值得关注：Agent 使用成本是开发者第一痛点。Codebase Memory 解决”Agent 记不住代码库”的问题，Token Diet 解决”Token 烧钱太快”的问题。两者互补。
  • Star/热度：📈 Token Diet ⭐596 | Codebase Memory MCP ⭐新项目 | C/TypeScript
  • 个人开发者切入点：
    • 集成打包为”Agent 效率订阅包”（记忆+省钱+调试），每月 $9-29
    • 基于此做企业版托管服务，按团队规模收费
    • 开发更多 Agent 优化 skill，在 Skills Market 售卖

💡 值得关注的趋势（3 个）

趋势 1：Agent 开发环境（ADE）军备竞赛

Orca、Excalibur、stablyai/orca、openai/codex-plugin-cc 同期爆发。2026 年 H2，Agent 正在从实验性质走向生产级，而 ADE 是兵家必争之地。

  • 启示：做 ADE 的插件/模板/工作流比做 ADE 本身更现实（竞争壁垒低）。Agent 技能市场、Agent 调试工具、Agent 成本优化工具是三个真切的空白。

趋势 2：小型专业化模型取代大模型预处理

Ternlight（7MB 嵌入，浏览器 WASM 运行）、Pulpie（网页清洗小模型）、OlmOCR（PDF 线性化工具链）——三者指向同一个方向：先用小模型做预处理，再用大模型做推理。

  • 启示：围绕这些工具做”数据处理 Pipeline as a Service”是明确的变现路径。RAG 预处理、PDF 结构化、网页清洗——每个都是独立的市场。

趋势 3：办公自动化 Agent 的 Zapier 时刻

OfficeCLI（Office 文件）、OpenConnector（1000+ SaaS 认证）、ChromeDevTools MCP（浏览器调试）、Page-Agent（GUI 操控）——Agent 正在获得操控物理世界的”手”。

  • 启示：垂直场景的 Agent 自动化模板（财务管理、合同处理、客户沟通）是个人开发者最容易切入的高毛利领域。无需自研底层，基于这些工具做编排即可。

🎯 行动建议

🎯 建议一：做 “Agent 效率工具包” 微 SaaS

思路：将 Codebase Memory MCP（代码记忆）+ Token Diet（省钱）+ Peek-CLI（浏览器调试）+ ChromeDevTools MCP 打包成一个订阅包。定价 $19/月（个人）或 $49/月（团队）。

为什么现在做：三个工具都是刚开源/刚出 beta，先发优势窗口约 2-3 个月。没有成熟的竞品。

执行路径：

  1. 用 1 周搭建集成 Web 界面（配置向导 + API Key 管理）
  2. 写 5 篇技术博客，投 Hacker News 和 Dev.to
  3. 前 100 用户终身 5 折，快速铺量

🎯 建议二：切入 “垂直行业 RAG 预处理” 赛道

思路：基于 Pulpie（网页清洗）+ OlmOCR（PDF 线性化）+ Ternlight（浏览器端嵌入），做法律/医疗/金融行业的文档预处理 API。收 API 调用费或按月订阅。

为什么现在做：RAG 是企业 AI 落地的主要形态，但数据预处理是最大痛点。“Put clean data in, get good answers out” 这个需求在 2026 年 H2 只会越来越迫切。

执行路径：

  1. 选一个细分行业（如合同审查），做端到端 demo
  2. 在 Product Hunt 发布 + 投稿 Indie Hackers
  3. 对接已有的法律科技/财税 SaaS 做集成合作

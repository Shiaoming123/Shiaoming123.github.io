  • 一句话：漂浮在屏幕上的 AI 助手，能看/听会议内容，屏幕共享时自动隐藏隐私

  • Star/热度：⭐ 481 | JavaScript
  • 个人开发者切入点：㈠ 提供云端服务版（多语言转录 + 摘要） ㈡ 做团队版 $15/月 ㈢ 作为 Obsidian/Notion 插件卖

  • Star/热度：⭐ 324 | Python

💡 值得关注的趋势（3 个）

趋势 1：AI 会议与屏幕感知工具爆发

cue 和类似产品的涌现说明”AI 能够在会议/屏幕共享中工作”成为新范式。隐私优先、本地运行是关键差异化。可做专注于特定场景（销售通话、编程会议）的版本。

趋势 2：编码 Agent 的记忆/上下文层成为基础设施

deja-vu 等工具试图解决 AI 编码”每次会话都失忆”的痛点。这是一个新兴基础设施层机会——类似数据库对 Web 应用的重要性。可围绕不同 Agent 做适配器。

趋势 3：AI + 垂直消费场景（时尚、穿搭、家居）

wardrobe 用 GPT vision 做衣橱管理，代表了”大模型视觉能力 + 日常生活场景”的创业方向。门槛低、用户需求真实、SaaS 模式成熟。

🎯 行动建议

基于今天的发现，个人开发者可以考虑的 2 个具体行动方向。

方向一：做 AI 会议摘要的隐私优先版

Otter.ai 验证了需求，但用户担心隐私。做一个本地处理（用 Ollama/本地模型）的会议摘要工具，每月 $10-15 的 SaaS 订阅。技术栈：Whisper（转录）+ 本地 LLM（摘要）。对标 cue 但更聚焦。

方向二：做垂直场景的 AI 视觉 SaaS

参考 wardrobe，选一个特定垂直场景（食谱识别、植物识别、宠物识别、家具布置），用 GPT-4o 视觉能力 + 简单 Web 界面，做专注的 $5/mo 小工具。开发成本 1-2 周，验证周期短。

🔥 今日精选（5 个最值得关注的）

1. shadcn/improve

  • 一句话：用最强模型审计代码库并生成优化计划，交给便宜模型执行 — 一个”模型路由+代码优化”管线
  • 为什么值得关注：shadcn 出品（shadcn/ui 作者），⭐6,274 快速飙升。解决了一个很实际的痛点：把昂贵的大模型（Claude Sonnet/GPT-5）当作”架构师”审计代码，把性价比低的模型（Claude Haiku/GPT-mini）当作”执行者”改代码，省 10 倍 token 成本
  • 个人开发者切入点：可以基于此思路做 SaaS 产品 — 提供代码审计+自动 PR 的订阅服务，面向中小团队。也可以 Fork 后添加自用 CI/CD 集成，作为开发流程加速器

2. diffusionstudio/lottie

  • 一句话：用 Claude Code 或 Codex 自然语言生成生产级 Lottie 动画
  • 为什么值得关注：⭐3,859，将 AI 对话直接转化为 Lottie 动画 JSON，设计师和开发者都能用。Lottie 动画是估值数十亿的市场（Airbnb 主导），覆盖 UI 动效、营销、游戏等领域
  • 个人开发者切入点：可以做”AI Lottie 动画工作室”SaaS（订阅制），或者作为 Figma 插件收费。门槛不高 — 用 Claude/Codex 就能快速构建类似功能

3. Omnigent

  • 一句话：开源多 AI Agent 编排框架，统一管理 Claude Code、Codex、Cursor 等编码 Agent
  • 为什么值得关注：⭐5,011，抓住了”Agent 碎片化”的痛点 — 现在开发者同时用多个 AI 编码工具，需要一个统一的编排层。支持沙箱、治理、多 Agent 协同
  • 个人开发者切入点：可以围绕 Omnigent 做企业级 Agent 治理面板（监控 token 消耗/权限/审计日志），或者做 Agent 技能市场（Marketplace），抽取交易佣金

4. WorkWeave Router（Show HN 136pts）

  • 一句话：智能模型路由工具，直接在 Claude、Codex 和 Cursor 中根据任务自动选择最佳模型
  • 为什么值得关注：HN 当日亮点，“模型路由”成为 2026 年下半年的新趋势 — 每个 AI 任务选最适合（性价比最高）的模型，而不是全用最贵的
  • 个人开发者切入点：同样思路可以做浏览器插件 / VS Code 插件 / API 网关。订阅模式：基础路由免费，高级规则和自定义模型池收费

5. baoyu-design（⭐2,025）

  • 一句话：在本地以 Agent Skill 方式运行 Claude Design，在 Cursor、Claude Code 等工具中直接生成 UI 原型、幻灯片和线框图
  • 为什么值得关注：把 Claude Design 从云端搬到本地，并且做成可复用的 Agent Skill。生成的 HTML/CSS 产物可直接交付
  • 个人开发者切入点：Skill 生态正在形成（BuilderIO/skills ⭐2,769 也印证了这一点）。做垂直领域 Agent Skill（落地页生成、简历设计、电商详情页），在 Skill 市场销售

💡 值得关注的趋势（3 个）

1. Agent Skill 生态崛起

  • 描述：BuilderIO/skills（⭐2,769）和 baoyu-design 等项目的爆发说明 Agent Skill 正成为新的开发范式。Claude Code、Codex、Cursor 都在生态化，允许第三方贡献可复用的技能包。Vercel 的 eve 框架（⭐2,669）也押注了这一方向
  • 对个人开发者的启示：Skill 市场就像当年 Chrome 插件市场 — 第一批做高质量垂直技能的人有望锁定收入。可做：SEO 审计 Skill、小红书文案生成 Skill、代码审查 Skill

2. “模型路由”成为新基础设施

  • 描述：shadcn/improve 和 WorkWeave Router 同时走红，加上 V2EX 上热议的”Agent 协作平台”，说明 AI 进入了”多模型协作”阶段。按任务分配 LLM 不再是企业特权，个人开发者也可以轻松实现
  • 对个人开发者的启示：可以做一个第三方”模型成本优化仪表盘” SaaS，帮助个人/小团队分析 LLM API 使用模式并推荐路由策略。月费 $9-29 的定价区间即可

3. macOS 开发者工具市场持续增长

  • 描述：claude-status-bar（⭐358）、wloc（Apple 网络定位修改工具，⭐454）以及 V2EX 上的”国区上架 2 天 / app 日活破 700”故事表明，macOS/iOS 小工具仍然有稳定的需求和付费意愿
  • 对个人开发者的启示：AI 相关的 macOS 菜单栏小工具（状态监控、快捷操作）是低门槛高回报的方向。可以考虑做：多 LLM 使用统计 dashboard、Prompt 历史管理器、模型切换开关

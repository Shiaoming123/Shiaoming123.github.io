AIGC 视频学习地图：从 Prompt 到分镜工作流

15 Jun, 2026
来源：wiki/courses/aigc/AIGC视频学习地图.md

这篇适合谁读

如果你刚开始学习 AI 视频生成，很容易把 prompt 写成一串“电影感、超真实、震撼”的形容词。真正稳定的做法，是把 prompt 写成导演能执行的分镜脚本：谁在做什么、为什么这么做、镜头怎么看、画面如何收尾。

这篇学习地图的目标不是复刻某个爆款短片，而是建立从想法到分镜、生成、筛选、剪辑和复盘的完整流程。

核心原则

AI 视频 prompt 应该像导演台本，而不是抽象愿望。

一个可执行 prompt 至少包含：

  • 时间、地点、主体。
  • 场景中可见的物件。
  • 主体动作和动作原因。
  • 景别、构图、运镜。
  • 材质、瑕疵、光线和真实感锚点。
  • 声音或静音限制。
  • 结尾留白，方便剪辑。

四个学习阶段

阶段 1：区分形容词和可执行信息

坏 prompt：

它的问题不是短，而是没有可执行信息：没有地点、动作、镜头、材质、声音和结尾。

改写方向：

  • 地点：废弃便利店、雨夜停车场、海边加油站。
  • 主体：旧式收银机器人，LED 表情屏，关节有油污。
  • 动机：听到货架后方异响，误判顾客身份。
  • 镜头：空镜、入场、中景、特写、反应。
  • 真实感：手持轻微晃动、低饱和、金属划痕、环境声。

阶段 2：写 10-15 秒单段视频

先不要追求长片。每次只练一个主体、一个地点、一个动作。

练习题：

  • 机器人夜班店员误认顾客。
  • 废土快递员骑旧摩托穿过沙尘。
  • 机械僧人在雨夜修复霓虹佛灯。

阶段 3：拆 4-6 个分镜

一个小段落可以这样组织：

分镜作用
1建立环境，让观众知道在哪里
2主体入场，明确人物或物体
3触发意外，制造动作原因
4展示反馈，体现物理真实感
5角色反应或反差结尾

阶段 4：剪成 30-60 秒短片

AI 视频不是一次生成长片，而是多镜头抽卡和后期拼接。要接受局部失败，用剪辑规避穿帮。

重点能力：

  • 统一角色和美术设定。
  • 统一色调、光线方向和镜头节奏。
  • 为每个镜头设计可剪点。
  • 记录失败镜头的原因，下一轮 prompt 才能变好。

Prompt 模板

版权和来源边界

公开案例可以学习结构，但不要照搬原始 prompt、角色名、IP 名、名人名和商业品牌。更稳的做法是把参考拆成抽象元素：

  • 不写具体 IP，写风格结构。
  • 不复制原 prompt，提炼镜头和动作方法。
  • 商业使用前检查素材来源、模型条款和音乐授权。
  • 在自己的笔记里保留来源链接，公开文章里明确学习边界。

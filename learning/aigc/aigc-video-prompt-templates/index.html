AIGC 视频 Prompt 模板库

15 Jun, 2026
来源：wiki/courses/aigc/AIGC视频Prompt模板库.md

这篇适合谁读

这篇适合想系统学习 AIGC 视频创作的人阅读。重点不是堆提示词，而是把来源治理、分镜结构、镜头语言、版权边界和可复用工作流整理成可练习的方法。

使用原则

这些模板用于写自己的视频，不用于照搬《丧尸清道夫》原始提示词。

每次只改一个层级：

  1. 先定主体、地点和动作。
  2. 再定风格和画质。
  3. 最后补分镜和声音。

不要一开始就堆满关键词。关键词越多，模型越难判断主次。

模板 1：10-15 秒多分镜叙事

适合练习短片片段、剧情钩子、反差小段落。

模板 2：单镜头一镜到底

适合变身、展示、武器充能、角色入场。

模板 3：动作戏

适合枪战、追逐、近战、怪物冲突。

模板 4：氛围和画质片段库

原子朋克 / 复古未来

可用元素：

  • 60 年代复古未来主义。
  • 圆润金属外壳。
  • 老式仪表、拨杆、铆钉、磨损喷漆。
  • 温暖夕阳与海盐蓝阴影。
  • 荒诞但有实拍质感。

末日废土

可用元素：

  • 低饱和沙尘。
  • 锈蚀车辆、废弃加油站、破碎霓虹牌。
  • 风吹过塑料布、纸屑、烟尘。
  • 人物服装有灰尘、汗渍、划痕。

赛博雨夜

可用元素：

  • 霓虹反光、湿地面、雨滴打在镜头前景。
  • 高对比暗部，局部彩色光源。
  • 手持跟拍或低角度推镜。

纪实实拍

可用元素：

  • 自然光。
  • 镜头不完全稳定。
  • 画面有轻微噪点和曝光不均。
  • 主体动作不过度戏剧化。

自检清单

生成前检查：

  • 是否有时间、地点、主体。
  • 是否有材质和瑕疵。
  • 是否说明声音。
  • 是否写了景别、构图、运镜。
  • 动作是否包含“为什么”。
  • 是否有物理反馈。
  • 是否避免了未经授权 IP 名和名人名。
  • 是否给结尾留了剪辑空间。
  • 是否总时长合理，单镜头不要过长。
  • 是否能拆成多个镜头抽卡。

改写流程

拿到一个糟糕 prompt：

改写步骤：

  1. 定具体地点：废弃便利店、雨夜停车场、海边别墅、加油站。
  2. 定主体材质：金属机器人、LED 表情屏、磨损外壳、关节油污。
  3. 定动作因果：听到异响、误判顾客、保护收银台、被滑倒的物件干扰。
  4. 定镜头：空镜、入场、特写、反应、收尾。
  5. 定真实感：摄影机、低饱和、手持微抖、环境声。

最后得到的不是“更长的 prompt”，而是“更可执行的分镜”。

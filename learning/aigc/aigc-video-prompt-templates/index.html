AIGC 视频 Prompt 模板库

15 Jun, 2026
来源：wiki/courses/aigc/AIGC视频Prompt模板库.md

这篇适合谁读

这篇适合想系统学习 AIGC 视频创作的人阅读。重点不是堆提示词，而是把来源治理、分镜结构、镜头语言、版权边界和可复用工作流整理成可练习的方法。

使用原则

这些模板用于写自己的视频，不用于照搬《丧尸清道夫》原始提示词。

每次只改一个层级：

  1. 先定主体、地点和动作。
  2. 再定风格和画质。
  3. 最后补分镜和声音。

不要一开始就堆满关键词。关键词越多，模型越难判断主次。

模板 1：10-15 秒多分镜叙事

适合练习短片片段、剧情钩子、反差小段落。

【基础设定】
时间：{时间段}
地点：{具体地点，加入可见物件}
主体：{角色/物体，外观、材质、状态}
环境：{天气、光线、动态元素}
声音限制：{环境声/动作声/是否禁止背景音乐}

【氛围画质】
风格核心：{3-5 个风格标签}
真实感：{摄影机/镜头/焦段/胶片颗粒/轻微手持}
材质细节：{磨损、灰尘、污渍、划痕、暗部细节}
限制词：避免 {游戏 CG 感、塑料感、过度磨皮、肢体扭曲、镜头漂移}
色彩影调：{低饱和/青橙/冷暖对比/复古暖色}

【画面内容】
总时长：约 {10-15} 秒，共 {4-6} 个分镜。

分镜 1：
景别：{全景/中景/特写}
构图：{中心/三分线/前景遮挡/对角线}
运镜：{固定/缓慢推进/跟拍}
内容：{建立环境，写一个可听见或可看见的细节}

分镜 2：
景别：{景别}
构图：{构图}
运镜：{运镜}
内容：{主体动作 + 为什么这样做 + 画面反馈}

分镜 3：
景别：{景别}
构图：{构图}
运镜：{运镜}
内容：{冲突或意外，写清物理反馈}

分镜 4：
景别：{景别}
构图：{构图}
运镜：{运镜}
内容：{角色反应或反差结尾，避免过度爆炸和强光}

模板 2：单镜头一镜到底

适合变身、展示、武器充能、角色入场。

核心主题：{题材} | {美学风格} | {真实感锚点} | {情绪}

【人物与基础设定】
主体：{人物/机器人/怪物/载具}
外观：{材质、服装、瑕疵、关键符号}
场景：{地点、天气、背景动态}

【氛围与画质】
视觉基调：模拟 {摄影机} + {镜头}，{焦段}，轻微手持浮动。
色彩：{主色调}，暗部保留细节，适度胶片颗粒。
限制：避免塑料感、游戏 CG 感、动作僵硬、过度干净。

【运镜规则】
单镜头一镜到底，无剪辑。
镜头从 {起始角度/景别} 开始，以 {速度} {推进/环绕/后拉/跟拍}。

【时间轴】
0-3 秒：{动作启动，写动作原因}
3-6 秒：{主体变化或动作升级，写物理反馈}
6-10 秒：{高潮动作，写材质、声音、环境反应}
10-15 秒：{收尾，留白，不堆过度特效}

模板 3：动作戏

适合枪战、追逐、近战、怪物冲突。

【基础设定】
地点：{可发生动作的具体空间}
主体 A：{外观、武器、状态}
主体 B：{数量、威胁方式、运动方式}
空间限制：{障碍物、狭窄通道、桌椅、车辆、雨水等}
声音限制：只保留 {脚步、金属碰撞、枪声、喘息、环境声}

【氛围画质】
风格核心：{电影动作 / 末日 / 黑色电影 / 赛博朋克}
真实感：{摄影机/镜头/手持/颗粒/暗部}
限制词：避免动作跳帧、肢体扭曲、无重量感、镜头漂移。

【画面内容】
分镜 1：建立空间和双方位置。
分镜 2：第一下动作必须有准备动作。
分镜 3：击中后要有受力反馈，不要直接消失。
分镜 4：用环境物件参与动作，例如桌角、玻璃、车门、积水。
分镜 5：收尾时保留一个安静或荒诞反应，方便剪辑。

模板 4：氛围和画质片段库

原子朋克 / 复古未来

可用元素：

  • 60 年代复古未来主义。
  • 圆润金属外壳。
  • 老式仪表、拨杆、铆钉、磨损喷漆。
  • 温暖夕阳与海盐蓝阴影。
  • 荒诞但有实拍质感。

末日废土

可用元素：

  • 低饱和沙尘。
  • 锈蚀车辆、废弃加油站、破碎霓虹牌。
  • 风吹过塑料布、纸屑、烟尘。
  • 人物服装有灰尘、汗渍、划痕。

赛博雨夜

可用元素：

  • 霓虹反光、湿地面、雨滴打在镜头前景。
  • 高对比暗部，局部彩色光源。
  • 手持跟拍或低角度推镜。

纪实实拍

可用元素：

  • 自然光。
  • 镜头不完全稳定。
  • 画面有轻微噪点和曝光不均。
  • 主体动作不过度戏剧化。

负面约束写法

不要只写“不要崩”。要明确避免哪些可见问题：

避免游戏 CG 感、塑料材质、过度磨皮、肢体扭曲、关节反向、无重量感动作、镜头无目的漂移、主体身份漂移、过度干净的场景、突然强光遮挡主体。

如果平台有独立 negative prompt 字段，写成名词短语更稳：

plastic skin, game CGI, twisted limbs, floating camera, identity drift, over-smoothed face, extra fingers, broken joints

自检清单

生成前检查：

  • 是否有时间、地点、主体。
  • 是否有材质和瑕疵。
  • 是否说明声音。
  • 是否写了景别、构图、运镜。
  • 动作是否包含“为什么”。
  • 是否有物理反馈。
  • 是否避免了未经授权 IP 名和名人名。
  • 是否给结尾留了剪辑空间。
  • 是否总时长合理，单镜头不要过长。
  • 是否能拆成多个镜头抽卡。

改写流程

拿到一个糟糕 prompt：

做一个很酷的末日机器人打丧尸视频，电影感，超真实。

改写步骤：

  1. 定具体地点：废弃便利店、雨夜停车场、海边别墅、加油站。
  2. 定主体材质：金属机器人、LED 表情屏、磨损外壳、关节油污。
  3. 定动作因果：听到异响、误判顾客、保护收银台、被滑倒的物件干扰。
  4. 定镜头：空镜、入场、特写、反应、收尾。
  5. 定真实感：摄影机、低饱和、手持微抖、环境声。

最后得到的不是“更长的 prompt”，而是“更可执行的分镜”。

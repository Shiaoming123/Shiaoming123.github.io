Mx-Shell《丧尸清道夫》方法论

15 Jun, 2026
来源：wiki/courses/aigc/Mx-Shell丧尸清道夫方法论.md

这篇适合谁读

这篇适合想系统学习 AIGC 视频创作的人阅读。重点不是堆提示词，而是把来源治理、分镜结构、镜头语言、版权边界和可复用工作流整理成可练习的方法。

一句话

把 AI 视频 prompt 写成“分镜导演台本”，而不是写成“风格形容词愿望清单”。

一个可用镜头至少要回答：

  • 画面里是谁。
  • 他在哪里。
  • 他正在做什么。
  • 为什么这么做。
  • 镜头怎么拍。
  • 画面有什么真实世界的瑕疵和物理反馈。
  • 声音和节奏如何服务剪辑。

两套结构：5 段式与三大段

公开材料里可以抽象出两套结构。它们不矛盾，只是粒度不同。

5 段式结构

适合写完整、稳定、可复用的 prompt：

  1. 核心主题

    • 3-6 个标签，确定题材和审美锚点。
    • 示例方向：原子朋克、末日丧尸、电影级质感、废土喜剧、真人实拍感。

  2. 人物与基础设定

    • 主体、外观、服装、材质、场景、环境动态。
    • 重点不是“帅”，而是具体：哑光金属、磨损边缘、油污关节、LED 表情屏。

  3. 氛围与画质

    • 用真实摄影语言替代空泛词。
    • 记录摄影机、镜头、焦段、光圈、色调、胶片颗粒、暗部细节。

  4. 运镜规则

    • 说明景别、角度、运动、速度和手持呼吸感。
    • “轻微手持浮动”是降低 CG 味的重要手段。

  5. 分镜时间轴

    • 单镜头按秒切片。
    • 多镜头按分镜拆：景别、构图、运镜、画面内容。

三大段结构

适合快速起草和自检：

  1. 基础设定

    • 时间、地点、人物、参考图描述、声音限制。

  2. 氛围画质

    • 风格核心、去 AI 味关键词、限制词、视觉基调、色彩影调。

  3. 画面内容

    • 分镜、景别、构图、运镜、故事内容。

怎么结合

实践中可以先用三大段快速写，再用 5 段式补细节：

  • 三大段保证不漏基本信息。
  • 5 段式保证镜头语言足够可执行。

好 prompt 的核心不是“华丽”，是因果

动作不能只写结果，要写动作链。

弱写法：

  • 机器人和丧尸打斗，动作很帅。

强写法：

  • 机器人先后撤半步，是因为听到吧台下方传来异响。
  • LED 表情从绿色笑脸切成黄色惊吓，是因为它误判环境安全。
  • 它抬枪前先低头确认弹巢，是为了让机械动作有真实准备过程。
  • 子弹击中后，丧尸身体不是直接消失，而是受冲击后肩膀后仰、脚步打滑、撞到桌角。

这类“为什么”和“物理反馈”会让模型更容易生成有连续性的动作。

去 AI 味的关键

1. 少用空泛赞美

少写：

  • 电影级
  • 震撼
  • 超高质量
  • 史诗感
  • 4K

这些词可以有，但不能单独支撑画面。

多写：

  • 模拟真实摄影机和镜头。
  • 低饱和色调。
  • 暗部保留细节。
  • 轻微胶片颗粒。
  • 手持拍摄的细小浮动。
  • 材质磨损、划痕、污渍和不对称。

2. 描述瑕疵

AI 很容易生成过度干净、过度平滑的画面。真实感通常来自瑕疵：

  • 金属边缘掉漆。
  • 关节处有油污。
  • 衣物有褶皱和灰尘。
  • 玻璃上有指纹和裂痕。
  • 皮肤或外壳保留轻微缺陷。
  • 光线不完美，暗部有压缩和噪点。

3. 镜头要有呼吸感

完全平滑的镜头容易像游戏 CG。轻微手持、缓慢推进、应激性微颤、呼吸般浮动，会让画面更接近实拍。

4. 声音要明确

如果平台会自动配乐，要明确声音限制：

  • 不要背景音乐。
  • 只保留环境声、机械声、脚步声、玻璃碎裂声。
  • 如果有台词，写具体语气和发生时机。

多分镜写法

每个分镜至少包括四项：

  • 景别：特写、中景、全景、大全景等。
  • 构图：中心构图、三分线、对角线、前景遮挡、过肩等。
  • 运镜：固定、缓慢推进、跟拍、环绕、拉远、甩镜等。
  • 画面内容：动作、原因、反馈、声音、表情变化。

推荐顺序：

  1. 空镜或环境建立。
  2. 主体入场。
  3. 异常或冲突出现。
  4. 主体做出反应。
  5. 反差、笑点、危险或留白结尾。

抽卡与剪辑现实

《丧尸清道夫》这类作品不能按“一条 prompt 一次生成完整短片”理解。更合理的心智模型：

  • 每个镜头都是候选池。
  • 生成多次，只挑能用片段。
  • 好镜头可能只用其中 1-3 秒。
  • 长片靠分镜、筛选、剪辑、调色和节奏统一。
  • 局部穿帮可以通过剪辑规避。

因此，prompt 写作也要服务剪辑：

  • 镜头开头和结尾要有可剪点。
  • 动作方向要尽量一致。
  • 光线方向和色调要统一。
  • 分镜之间要有动作或声音上的连接。

IP 与平台拦截

原始提示词里保留了部分 IP 名和名人风格引用，用于忠实记录公开材料。但实际创作时应尽量改成通用描述：

  • 不写具体 IP 名，改写为题材、美术、材质、时代感。
  • 不写名人名字，改写为舞蹈风格、节奏、身体动作。
  • 商业用途必须避开未授权 IP。

可迁移的创作公式

题材公式：

一个反差主体 + 一个强风格场景 + 一个具体动作困境 + 一个情绪/喜剧/危险反应 + 一个留白结尾

Prompt 公式：

基础设定：谁 / 何时 / 何地 / 什么材质 / 声音限制
氛围画质：真实摄影语言 / 色彩 / 质感 / 限制词
画面内容：分镜列表，每镜写景别、构图、运镜、动作因果、物理反馈

复盘公式：

这一镜想表达什么？
生成结果哪里像 AI？
问题来自主体设定、动作因果、镜头语言、材质、声音，还是时长过长？

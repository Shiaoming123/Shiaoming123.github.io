AIGC 国风武侠打斗 Prompt 拆解

目前没有找到成熟的“雾山五行逐镜头拉片 + 系统 prompt 库 + skill”的开源项目。

15 Jun, 2026
来源：wiki/courses/aigc/AIGC国风武侠打斗Prompt拆解.md

这篇适合谁读

这篇适合想系统学习 AIGC 视频创作的人阅读。重点不是堆提示词，而是把来源治理、分镜结构、镜头语言、版权边界和可复用工作流整理成可练习的方法。

结论

目前没有找到成熟的“雾山五行逐镜头拉片 + 系统 prompt 库 + skill”的开源项目。

但已经有几类可用材料：

  • 单条高相关 prompt：OpenNana 的水墨武侠激战。
  • 通用动作库：Seedance prompt vault 里的 Action & Fight 分类。
  • 生成型 skill：Seedance prompt skills，可按自然语言生成仙侠/动作 prompt。
  • 平台语法 skill：支持 @视频 参考动作、运镜、节奏。

最佳策略不是复制“雾山五行”，而是抽象成“国风硬边水墨动作体系”。

需要抽象的核心特征

1. 画风层

不要直接依赖作品名，改写为可迁移的视觉语言：

  • 硬边水墨线条。
  • 高对比红黑 / 黑白灰 + 单一强调色。
  • 墨气、飞白、速度线。
  • 爆裂但清晰的剪影。
  • 国漫画风但动作读得清。
  • 夸张动态构图，身体姿态有张力。

2. 动作层

武侠打斗 prompt 最容易失败在“动作太快但不清楚”。要写清每次攻防的动作因果：

  • 起手：谁先发力，从哪里发力。
  • 进攻：劈、砍、刺、挑、扫、砸、格挡、反击。
  • 接触：武器相撞、拳脚命中、身体偏移。
  • 反馈：火花、墨气、尘土、地面裂纹、衣摆甩动。
  • 收束：双方拉开、背对背落地、武器相抵僵持。

3. 运镜层

高燃武侠动作不是一直快。建议快慢交替：

  • 低角度高速跟拍：用于冲刺和接近。
  • 快速切镜：用于攻防互换。
  • 慢动作特写：用于击中或武器碰撞瞬间。
  • 环绕运镜：用于双方势均力敌、气场对峙。
  • 仰视全景：用于大招对轰。
  • 中景慢推：用于僵持和留白结尾。

4. 节奏层

推荐 15 秒五段式：

  1. 0-2 秒：无铺垫，直接进入武器对撞。
  2. 2-5 秒：高速攻防互换。
  3. 5-8 秒：近身连击和物理反馈。
  4. 8-12 秒：大招或气场爆发。
  5. 12-15 秒：僵持、定格、留白。

如果模型动作崩，就改成 8-10 秒三段式：

  1. 0-3 秒：对峙到冲刺。
  2. 3-7 秒：一次清晰交锋。
  3. 7-10 秒：落地收束。

来源拆解

OpenNana 水墨武侠激战

链接：https://opennana.com/awesome-prompt-gallery/fog-hills-ink-martial-arts-battle

可学习点：

  • 用五个镜头建立完整战斗段落。
  • 每个镜头都有动作主题：对撞、攻防、近身、大招、僵持。
  • 强调双方势均力敌，避免一方碾压导致动作单调。
  • 有镜头语言：低角度、高速跟拍、环绕、慢动作、仰视全景、慢推。

需要改写的地方：

  • 删除具体作品名。
  • 把“同款画风”改为“硬边水墨国风动画质感”。
  • 把“极致”“拉满”等泛化词换成可见细节。

HuyLe82US Action & Fight / Bamboo Run

链接：https://github.com/HuyLe82US/awesome-seedance-prompts

可学习点：

  • 竹林、雨声、压迫氛围适合作为武侠场景。
  • 打斗前用安静建立张力，再用雷声作为动作触发点。
  • 武器碰撞后写环境反馈：雨滴爆开、竹叶被剑气切开、地面泥水飞溅。
  • 背对背落地是很好的收束镜头。

需要改写的地方：

  • 把英文 prompt 改成中文结构化分镜。
  • 把“一段大动作”拆成多个 3 秒左右片段。

Seedance prompt skills

相关来源：

  • https://github.com/songguoxs/seedance-prompt-skill
  • https://github.com/dexhunter/seedance2-skill

可学习点：

  • 10 秒以上用时间戳分镜。
  • 每段写画面、运镜、音效。
  • 参考视频时要明确：只参考动作编排、运镜节奏、打击感，不复制角色和美术。

失败模式

动作糊成一团

原因：

  • 同一镜头里塞了太多招式。
  • 全程高速，没有慢动作节点。
  • 没写接触点和受力反馈。

修正：

  • 每 3 秒只写 1 个动作重点。
  • 把击中瞬间改成慢动作特写。
  • 明确“武器相撞 -> 火花/墨气 -> 双方后退/偏移”。

像游戏特效，不像国风动画

原因：

  • 只写了“高燃、4K、酷炫”。
  • 没有线条、墨气、留白、色彩约束。

修正：

  • 加硬边水墨线条、飞白、墨雾、单一强调色。
  • 减少全屏发光和爆炸。

角色变成 IP 复刻

原因：

  • 直接写作品名或角色名。
  • 上传强识别参考图。

修正：

  • 改用原创角色。
  • 参考视频只声明“动作/运镜/节奏”，排除角色和美术。

后续收集方向

  • 找更多“动作分镜拆解”而不是单句 prompt。
  • 优先收集带时间轴、运镜、音效、失败修正的来源。
  • 对雾山五行、鬼灭、火影等强 IP，只记录抽象风格和动作原则，不保存可直接复刻的侵权表达。

AIGC 视频 Prompt 来源库

以后需要找 AIGC 视频 prompt、动作 prompt、武侠 prompt、Seedance prompt、随机 prompt 灵感时，优先从这个页面检索，而不是每次从零开始搜网页。

15 Jun, 2026
来源：wiki/courses/aigc/AIGC视频Prompt来源库.md

这篇适合谁读

这篇适合想系统学习 AIGC 视频创作的人阅读。重点不是堆提示词，而是把来源治理、分镜结构、镜头语言、版权边界和可复用工作流整理成可练习的方法。

用途

以后需要找 AIGC 视频 prompt、动作 prompt、武侠 prompt、Seedance prompt、随机 prompt 灵感时，优先从这个页面检索，而不是每次从零开始搜网页。

使用顺序：

  1. 先查本页面的“高优先级来源”。
  2. 如果是武侠、国风、水墨、打斗，先看 AIGC 国风武侠打斗 Prompt 拆解。
  3. 如果是通用短片 prompt，先看 Mx-Shell《丧尸清道夫》方法论 和 AIGC 视频 Prompt 模板库。
  4. 如果本页没有，再联网搜索，并把新来源补回本页。

来源评级

评级标准：

  • A：可作为长期来源库或方法论入口，值得反复查。
  • B：有可用 prompt 或案例，但需要筛选、改写或注意版权。
  • C：只是线索、二手转述、营销页或单条 prompt，适合发现灵感，不适合作为最终依据。

高优先级来源

1. jnMetaCode/ai-shortfilm-prompts

链接：https://github.com/jnMetaCode/ai-shortfilm-prompts

评级：A

用途：

  • 学 Mx-Shell《丧尸清道夫》公开方法论。
  • 查 AI 短片 5 段式 prompt 结构。
  • 查多分镜叙事、变身、机甲、打斗、赛博武侠等案例。

适用任务：

  • 写电影感短片 prompt。
  • 把想法拆成分镜。
  • 降低 AI 味、塑料感和 CG 感。
  • 做自己的 AIGC 视频 skill。

边界：

  • 方法论、模板、Skill 可复用。
  • prompts/ 里的原始提示词归 Mx-Shell，学习结构，不要公开搬运全文或商业再发布。

2. OpenNana 雾山五行风格燃向水墨武侠激战

链接：https://opennana.com/awesome-prompt-gallery/fog-hills-ink-martial-arts-battle

评级：B+

用途：

  • 当前最直接命中“雾山五行 / 水墨武侠 / 激烈打斗 prompt”的页面。
  • 已拆成五个镜头：开场武器对撞、高速攻防、近身搏杀、大招对轰、僵持定格。

适用任务：

  • 学国风水墨动作 prompt 的镜头结构。
  • 提炼“武器碰撞、墨气、速度线、冲击波、势均力敌”的表达。

边界：

  • 页面直接使用“雾山五行”字样。用于学习和私下测试可以，正式产出应改成“硬边水墨国风动画、凌厉线条、高对比红黑色调”等通用描述。

3. HuyLe82US/awesome-seedance-prompts

链接：https://github.com/HuyLe82US/awesome-seedance-prompts

评级：A-

用途：

  • Seedance prompt vault，包含 Action & Fight 分类。
  • 其中 Bamboo Run: Duel at Dusk 是竹林武侠决斗案例，适合学习“压迫氛围 -> 同时冲刺 -> 武器交锋 -> 竹叶/雨水受力反馈 -> 背对背落地”的动作结构。

适用任务：

  • 找动作 / 打斗 / 竹林武侠 / 高速运镜 prompt。
  • 抽象动作场景的镜头节奏和物理反馈。

边界：

  • 部分案例来自 X 用户，二次使用要保留来源意识。学习结构，改写角色和场景。

4. songguoxs/seedance-prompt-skill

链接：https://github.com/songguoxs/seedance-prompt-skill

评级：A-

用途：

  • Claude Code Skill，专为 Seedance 2.0 生成视频提示词。
  • 示例包含“仙侠战斗 15 秒横屏”，按时间段写动作、镜头、音效。

适用任务：

  • 写中文 Seedance prompt。
  • 参考 skill 的交互流程：确认时长、比例、参考素材、风格偏好，再给 2-3 个版本。

边界：

  • 可作为 skill 结构参考，不必照搬其示例内容。

5. MapleShaw/seedance2.0-prompt-skill

链接：https://github.com/MapleShaw/seedance2.0-prompt-skill

评级：A-

用途：

  • 更系统的 Seedance 2.0 Skill。
  • 包含相机四维编码、长视频生产流水线、素材剪辑节奏、角色卡/首帧图/关键帧图提示词规范。

适用任务：

  • 做长视频分段。
  • 设计角色卡、关键帧、首尾帧。
  • 处理素材剪辑和风格统一。

边界：

  • 更偏平台工程化，不是武侠专用。

6. dexhunter/seedance2-skill

链接：https://github.com/dexhunter/seedance2-skill

评级：A-

用途：

  • Seedance 2.0 多模态 prompt skill。
  • 重点讲 @图片、@视频、@音频 引用语法。
  • 对“上传武侠打斗参考视频，只复刻运镜和动作节奏”很有价值。

适用任务：

  • 根据参考图保持角色一致。
  • 根据参考视频复刻动作、运镜、节奏。
  • 根据音频做卡点或音效控制。

边界：

  • 如引用受版权保护的视频，正式产出要避免复刻具体角色、镜头和可识别 IP 表达。

大型 prompt 池

7. YouMind-OpenLab/awesome-seedance-2-prompts

链接：https://github.com/YouMind-OpenLab/awesome-seedance-2-prompts

评级：B+

用途：

  • 大规模 Seedance prompt 库，号称 2000+ prompt。
  • 覆盖 cinematic、anime、UGC、ads、meme 等。

适用任务：

  • 随机找灵感。
  • 搜 action、anime、wuxia、fight、battle 等关键词。
  • 批量观察不同类型 prompt 的结构。

边界：

  • 量大但噪声也高，需要人工筛。

8. HiAPIAI/awesome-seedance-2-0-prompts

链接：https://github.com/HiAPIAI/awesome-seedance-2-0-prompts

评级：B

用途：

  • 163 个 Seedance 2.0 视频 prompt 示例。
  • 带 OpenAI-compatible HiAPI 请求格式，适合 agent 或 API 集成参考。

适用任务：

  • 查可 API 化的 prompt 写法。
  • 看 prompt 与生成结果如何组织成数据集。

边界：

  • 不是武侠专项，按类别筛选。

9. Atlas Cloud Seedance 2 Prompt Library

链接：https://www.atlascloud.ai/prompts-hub/seedance-2-prompt

评级：B

用途：

  • 线上 prompt gallery，显示有 1000+ Seedance prompt。
  • 有 Action、Anime、Cinematic 等分类。

适用任务：

  • 快速找 action / anime / fantasy / cinematic 灵感。
  • 查参考资产 @mentions 的使用方式。

边界：

  • 网页库，非 GitHub 项目；需要手工筛选和记录来源。

官方或半官方指南

10. HunyuanVideo 1.5 Prompt Handbook

链接：https://github.com/Tencent-Hunyuan/HunyuanVideo-1.5/blob/main/assets/HunyuanVideo_1_5_Prompt_Handbook_EN.md

评级：A-

用途：

  • 腾讯混元视频官方 prompt handbook。
  • 公式包含主体、动作、场景、景别、运镜、灯光、风格、氛围。

适用任务：

  • 建立跨模型通用 prompt 框架。
  • 把 Seedance 经验迁移到 HunyuanVideo 或其他视频模型。

边界：

  • 英文官方指南，模型特性不同，不能直接套 Seedance 参数。

11. 阿里云百炼 / 万相 文生视频 Prompt 指南

链接：https://help.aliyun.com/zh/model-studio/text-to-video-prompt

评级：A-

用途：

  • 官方文生视频 / 图生视频 prompt 指南。
  • 适合建立最基础的结构化公式：主体、情境、运动。

适用任务：

  • 初学者 prompt 基础。
  • 对照不同平台的 prompt 结构差异。

边界：

  • 偏基础和平台文档，不提供复杂武侠打斗拆解。

低优先级或只作线索

短视频平台、X、公众号、新闻拆解

评级：C

用途：

  • 找热门案例、爆款方向、短期灵感。
  • 例如“AI 视频打斗画面软绵无力”“Seedance 2.0 测评”“国漫水墨打斗热梗视频拆解”等。

边界：

  • 多为二手转述或营销内容。
  • 可能含未授权 IP、热梗、搬运 prompt。
  • 不作为最终事实依据，不直接复制。

使用守则

  • 从来源库找结构，不照搬 IP。
  • 优先抽象成“动作、运镜、材质、节奏、物理反馈”。
  • 对明确带有“雾山五行”等作品名的 prompt，改成通用风格描述。
  • 对社交平台 prompt，默认只当线索，除非能追到原作者或原始仓库。
  • 生成自己的作品时建立原创角色、原创场景和原创招式。

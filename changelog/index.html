维护日志

记录这个博客站点的设计改进、内容新增、部署调整和后续维护。

  1. 2026-06-28

    content

    Arena 大模型排行榜专栏

    新增 /arena/ 专门页面，把 Arena 官方 Text Arena Overall 榜单快照整理成科幻 HUD 风格的大模型观察入口。

    • 顶部导航新增 Arena 入口，并补齐中英文页面标题、描述和面包屑标签。
    • 页面主体新增六维能力雷达图示例，用 Text、Code、Agent、Vision、Cost、Context 对头部模型做可视化对比。
    • 雷达图支持模型切换、高亮轮廓、能力条联动、扫描线和霓虹节点动效，形成类游戏科幻 HUD 观感。
    • 由于 Arena 官方页面拒绝第三方 iframe 嵌入，改为站内快照表格，避免发布一个无法加载的内嵌窗口。
  2. 2026-06-27

    design

    Demo 演示窗口全屏预览

    给 /demos/ 的内嵌演示窗口增加网页级全屏预览，减少固定 iframe 窗口对体验型项目的限制。

    • 每个演示 iframe 右上角新增“全屏预览”控制，可把当前 Demo 放大到整个网页视口。
    • 全屏状态下支持同一按钮或 Esc 退出，并锁定页面滚动，避免预览时背景内容干扰操作。
    • 保留原有“打开 Demo”和 GitHub 外链入口，跨域项目仍可通过固定预览或独立窗口访问。
  3. 2026-06-27

    design

    Paper Island 3D 互动叙事 Demo

    新增一个站内 WebGL 互动叙事原型，用轻量程序化几何复刻诗意 3D 体验的核心结构，而不复制原站素材或代码。

    • 新增 /demos/interactive-3d-story/ 独立体验页，使用 React Three Fiber 承载全屏 3D 场景。
    • 实现拖动开花、章节推进、镜头转场、纸纹漫画化视觉、WebAudio 环境音和移动端提示。
    • 将 Paper Island 接入 /demos/ 演示索引和 iframe 预览，作为后续 glTF 资产替换的 MVP 基线。
  4. 2026-06-27

    platform

    Pages CMS 内容后台接入

    为 Astro 源码博客接入 Git-backed 内容后台，让文章、学习地图和站点数据可以通过表单化后台维护。

    • 新增 Pages CMS 配置，开放普通文章、学习地图文章、关于页、学习聚类、Demo 项目和更新日志的后台编辑入口。
    • 将学习聚类、Demo 项目和更新日志迁移为 JSON 数据源，保留原有 TypeScript 导出接口，便于后台表单化管理。
    • 新增 GitHub Actions 发布流水线，通过专用 deploy key 将 Astro 构建产物推送到 GitHub Pages 发布仓。
  5. 2026-06-27

    content

    Open Design Gallery 接入演示页

    把个人整理的前端素材预览画廊接入博客演示栏目，让设计参考和项目 Demo 可以从同一个入口浏览。

    • 新增 Open Design Gallery 演示卡片，首页 Demo 区和 /demos/ 页面都会展示入口。
    • 在 /demos/ 页面内嵌 GitHub Pages 项目页，保留独立打开画廊和查看源码仓库的外链。
    • 避免占用 /open-design-gallery/ 项目页路径，防止和 GitHub Pages 项目站路由发生冲突。
  6. 2026-06-27

    design

    文章目录抽屉与阅读进度联动

    将桌面端本文目录升级为文章页内的可隐藏抽屉，并随阅读进度实时高亮当前小节。

    • 右侧目录不再作为展开的固定侧栏，而是文章页专属的常驻抽屉，支持一键收起和展开。
    • 目录链接会根据滚动位置同步当前小节状态，抽屉顶部显示文章阅读进度百分比。
    • 抽屉只在桌面端启用，移动端继续使用标题下方的折叠式本文导览，避免遮挡正文。
  7. 2026-06-27

    design

    文章导览与系列路线体验修正

    把文章右侧区域改回读者更熟悉的本文目录，同时将系列路线收束为标题下方的紧凑导览卡，保持正文居中阅读。

    • 桌面端右侧只显示本文目录，避免系列路线被误认为当前文章大纲。
    • 系列路线和同类延伸改为正文标题下方卡片，突出上一篇、下一篇和完整路线入口。
    • 移动端使用折叠式本文导览，目录和路线分组展示，减少首屏拥挤和横向溢出。
    • 文章 shell 重新以正文列居中，保留代码块、Mermaid、图片预览和标题锚点增强。

    content

    • 新增 /bookmarks/ 书签资源库页面，只展示经过二次筛选的公开技术、学习和创作资源。

    • 首页新增资源图谱入口，顶部导航增加“资源”，方便从主站进入公开资源库。

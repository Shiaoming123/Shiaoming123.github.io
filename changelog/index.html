维护日志

记录这个博客站点的设计改进、内容新增、部署调整和后续维护。

  1. 2026-06-26

    content

    Courses 学习内容批量扩容

    将知识库 courses 中剩余适合公开的课程笔记迁入学习地图，形成更完整的个人学习资料库。

    • 新增 31 篇学习地图文章，覆盖 AI 产品项目网络、AIGC 视频创作、Stanford CS 课程和微信小程序学习路线。
    • 新增“微信小程序”学习聚类，作为微信 AI 接入和轻量工具产品化的学习入口。
    • 迁入时清理 Obsidian 私链、本机路径、邮箱、私有 raw 路径和不适合公开的内部记录。
    • 新增可复跑的 courses 导入脚本，后续知识库课程更新时可以继续批量同步。
  2. 2026-06-26

    content

    首页最近更新接入学习地图

    首页 Index 的“最近更新”不再只展示普通文章，也会按更新时间混入学习地图内容。

    • 最近更新数据源改为 posts 与 learning 两个内容集合混排。
    • 学习地图文章复用首页文章卡片样式，链接指向对应的 /learning/ 页面。
    • 首页更新展示数量从 4 条调整为 8 条，避免新学习内容完全挤出普通文章。
  3. 2026-06-26

    maintenance

    Mermaid 流程图可读性优化

    针对学习地图文章中 Gantt 图和流程图过小的问题，扩大图表阅读区域，并增加全屏式放大预览。

    • Mermaid 图表默认突破正文窄栏，使用更宽的阅读容器展示复杂流程图。
    • 长 Gantt、timeline、journey 等图表支持横向滚动，避免被压缩到不可读。
    • 所有 Mermaid 图表增加“放大查看”按钮，可在独立预览层中查看细节。
    • 统一 Mermaid 字体大小和深浅色主题下的预览样式。
  4. 2026-06-26

    content

    学习地图与课程内容扩容

    把知识库 courses 中适合公开的学习路线整理为独立栏目，形成文章之外的系统化学习入口。

    • 新增顶部导航“学习”和 /learning/ 学习地图页面。
    • 新增 AI 产品路线、AIGC 创作、CS 课程图谱三个学习聚类。
    • 首批迁入并改写 6 篇课程笔记，清理内部路径、Obsidian 私链和不适合公开的求职自评内容。
    • 学习内容接入搜索、RSS 和百度 sitemap，保持 GitHub Pages 静态发布方案不变。
  5. 2026-06-25

    platform

    Markdown 增强、贡献图和首页动效

    补齐技术博客需要长期使用的内容展示能力，并让首页更像一份活的技术档案。

    • 增加 Mermaid 流程图渲染能力，支持在文章中直接写流程图、时序图和状态图。
    • 新增 DataChart、EmbedCard、GithubRepoCard 等 MDX 组件，用于数据图、外部页面嵌入和仓库展示。
    • 文章列表支持自动读取预览图，优先使用 frontmatter，缺省时读取正文第一张图或默认封面。
    • 首页加入 GitHub 贡献热力图和轻量 canvas 粒子层。
    • 关于页补充为职业档案式介绍，保留公开且不敏感的信息。
  6. 2026-06-24

    content

    Demo 演示栏目上线

    把已经部署到 Vercel 的项目嵌入个人博客，让项目复盘不只停留在文字描述。

    • 演示页使用宽屏 iframe 容器，保留打开 Demo 和 GitHub 的外链入口。
    • 首页增加 Demo 引导区，让读者可以从主线内容进入可体验项目。
